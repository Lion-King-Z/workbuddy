## WebSearch 信源分级搜索规则（v2.2 2026-06-30）

> ⛔ **强制性规则。** 所有投研Skill的WebSearch调用必须遵循以下优先级，不可跳过。

### 分级搜索策略

```
第一级（强制，B+级） — 优先 allowed_domains
  ├─ eastmoney.com     东方财富（公告原文/财联社快讯）
  ├─ stcn.com          证券时报
  ├─ cs.com.cn         中国证券报
  ├─ cninfo.com.cn     巨潮资讯（法定公告原文，A级）
  ├─ sse.com.cn        上交所（A级）
  └─ szse.cn           深交所（A级）

第二级（B级，种子补充）— 不加 domain 限制
  └─ 财联社/21世纪经济报道/界面新闻/第一财经 → B级

第三级（兜底，C级，仅线索）— 不做 domain 过滤
  └─ 百家号/搜狐/网易/知乎/雪球 → C级（⚠️ 不可直接引用，需交叉验证）
```

### 信源标注规则

| 信源 | 等级 | 可否直接引用 |
|------|:---:|:---:|
| 巨潮/上交所/深交所公告原文 | **A** | ✅ 直接引用 |
| 东方财富/证券时报/中证报 | **B+** | ✅ 直接引用，标注来源 |
| 浑水调研电话会纪要（可溯源原文） | **B** | ✅ 可引用 |
| 财联社/界面/第一财经 | **B** | ✅ 可引用 |
| 券商研报（非行业白皮书） | **C** | ⚠️ 需交叉验证 |
| 百家号/搜狐/网易（含AI生成内容） | **C-** | ⛔ 仅作线索，不对照不能入结论 |
| 雪球/知乎/微信群 | **D/E** | ⛔ 仅情绪温度计 |

### ⛔ C- 级信源禁令

百家号、搜狐、网易等平台的股票/产业类文章**默认 C- 级**，原因：
- 大量为AI批量生成，信息被稀释/扭曲
- 无编辑审校，不标注信息来源
- 不可追溯，出错后无法确认原始出处

**C- 级使用规则**：可作线索触发方向搜索，但**写入任何投研结论前必须**：
1. 回溯到第一级 allowed_domains 确认有无对应原始公告
2. 如有 → 替换为B+级信源
3. 如无 → 标注"C-级信源，仅线索，待验证"，不得写入紧缺度/评级判断
