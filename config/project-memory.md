## 项目文件结构（2026-06-14 精简版）

```
F:\workbuddy\锅师\
├── 深度研究/                          ← 产业链深度研报
│   ├── MMDD-MMDD/                    ← 周文件夹（按产出日期归入）
│   │   └── YYYY-MM-DD_主题深研.md
│   └── YYYY-MM_月报.md               ← 月报在深度研究根
├── 浑水调研/                          ← 浑水相关（电话会+景气度+紧缺度+月度跟踪）
│   ├── MMDD-MMDD/                    ← 周文件夹（同深度研究结构）
│   │   ├── 电话会_*.md
│   │   ├── 浑水研报紧缺总结_*.md
│   │   └── 供应紧缺度分析_*.md
│   ├── 浑水跟踪_YYYY-MM.md           ← 月度跟踪（跨周期，放根目录）
│   ├── 紧缺度追踪总表_YYYYMMDD.md    ← 跨周期追踪表
│   └── 紧缺信号时间线_*.html         ← 可视化
├── 信息源清单.md
├── 年报_YYYY.md
└── 聚宝盆索引.md

F:\个人成长\                           ← 独立文件夹（不在锅师项目内）
├── YYYYWww_个人成长.md               ← 思维提升+盲区+优化建议
└── YYYY-MM_个人成长.md               ← 月度认知复盘
```

### 核心原则
- **2目录+N文件**：深度研究/ + 浑水调研/ + 根目录独立文件
- **深度研究 = 产业链**：仅放产业链深度研报
- **浑水调研 = 浑水生态**：电话会、景气度扫描、紧缺度分析、月度跟踪——凡浑水相关全进此处
- **时间容器 > 类型容器**：按产出日期归入周文件夹
- **跨周期追踪表/时间线放根目录**：不拆入周文件夹
- **个人成长在项目外**：`F:\个人成长\` 独立管理
- **无形式化产出**：不生成日报/简报/研报，只产出电话会深度分析
- 管理 Skill：`.workbuddy/skills/项目结构规范/SKILL.md`
