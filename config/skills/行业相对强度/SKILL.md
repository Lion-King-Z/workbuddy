# 行业相对强度 Skill v1.0

**定位**：行业轮动信号扫描器。回答"钱在往哪个行业流，从哪个行业流出"。

**核心原则**：
1. 行业相对强度 = 行业指数涨跌幅 - 大盘涨跌幅
2. 连续多日跑赢大盘 = 资金流入信号
3. 连续多日跑输大盘 = 资金流出信号
4. 只做数据扫描和信号标注，不做行业推荐

---

## 一、触发条件

- 用户问"行业轮动""相对强度""板块强弱""哪个行业强"
- 每周研判调用（作为行业层方向判断输入）
- 窄门筛股调用（作为事件映射的行业背景）

---

## 二、数据获取

### 2.1 行业指数

通过 **tdx-connector** 获取申万一级行业指数（31个行业）：

```
tdx_quotes(code=行业指数代码)
→ 提取：最新价 / 近5日涨跌幅 / 近20日涨跌幅 / 近60日涨跌幅
```

若tdx-connector不返回行业指数，降级为 **akshare-stock** 的 `sw_index_daily` 接口。

### 2.2 大盘基准

- 沪深300（000300）或上证指数（000001）作为基准
- 同样获取近5日/20日/60日涨跌幅

### 2.3 数据频率

| 场景 | 频率 |
|:---|:---|
| 每周研判 | 每周一次（周日晚） |
| 窄门筛股 | 每日（如有事件映射需要） |
| 用户主动调用 | 按需 |

---

## 三、核心计算

### 3.1 相对强度计算

```
RS_5 = 行业近5日涨跌幅 - 大盘近5日涨跌幅
RS_20 = 行业近20日涨跌幅 - 大盘近20日涨跌幅
RS_60 = 行业近60日涨跌幅 - 大盘近60日涨跌幅

综合RS = RS_5 × 0.5 + RS_20 × 0.3 + RS_60 × 0.2
```

### 3.2 行业强度分级

| 综合RS | 强度等级 | 信号 |
|:---:|:---:|:---|
| > +10% | 🔴 强势 | 资金持续流入，主线方向 |
| +3% ~ +10% | 🟠 偏强 | 资金温和流入，值得关注 |
| -3% ~ +3% | 🟡 中性 | 无明显方向 |
| -10% ~ -3% | 🔵 偏弱 | 资金温和流出 |
| < -10% | ⚪ 弱势 | 资金持续流出，回避 |

### 3.3 轮动信号检测

| 信号类型 | 检测条件 | 含义 |
|:---|:---|:---|
| 🔥 加速流入 | RS_5 > RS_20 > RS_60 且 RS_5 > +5% | 短期资金加速流入 |
| 📈 持续流入 | RS_5 > 0 且 RS_20 > 0 且 RS_60 > 0 | 中长期资金持续流入 |
| ⚠️ 动能衰减 | RS_5 < RS_20 且 RS_20 > +5% | 短期动能减弱，可能见顶 |
| 🔄 轮动信号 | RS_5从负转正 且 RS_20 < 0 | 行业可能开始轮动切入 |
| 📉 持续流出 | RS_5 < 0 且 RS_20 < 0 且 RS_60 < 0 | 中长期资金持续流出 |
| 💤 底部企稳 | RS_60 < -10% 但 RS_5 > 0 | 长期弱势但短期企稳 |

## 五、与每周研判的衔接

每周研判 Step 1.5 宏观噪音过滤器可调用本Skill：
- 资金流维度：引用行业相对强度的"资金流入/流出行业"
- 跨市场联动：行业轮动方向验证宏观判断
- 权重分配器：强势行业对应的供给刚性类型权重上调

---

## 六、注意事项

- 行业指数以申万一级分类为准（31个行业）
- 相对强度是统计信号，不是投资建议
- 短期（5日）波动大，需结合20日/60日确认趋势
- 行业轮动有滞后性，信号出现后通常1-2周才反映到个股
- 严格遵循决策边界：只排序+标注信号，不推荐行业
