# K线频率参数映射表

在调用 `get_kline` 方法时，`frequency` 参数使用以下数值：

| frequency 值 | 对应周期     | 说明               |
|--------------|--------------|--------------------|
| 0            | 5分钟        | 5分钟K线           |
| 1            | 15分钟       | 15分钟K线          |
| 2            | 30分钟       | 30分钟K线          |
| 3            | 1小时        | 60分钟K线          |
| 4            | 日线         | （部分版本可用）   |
| 5            | 周线         | 周K线              |
| 6            | 月线         | 月K线              |
| 7            | 1分钟        | 1分钟K线           |
| 8            | 分笔成交     | 逐笔成交数据       |
| 9            | 日线         | **推荐使用**       |

**注意**：
- 最常用的是 `9`（日线），返回每日的OHLCV数据。
- 分钟线数据可能受服务器限制，部分周期不可用。
- 具体支持情况以通达信服务器实际返回为准。
