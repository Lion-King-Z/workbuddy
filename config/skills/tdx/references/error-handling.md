# 错误处理指南

## 常见错误及解决方案

### 1. 连接超时 / 无法连接到行情服务器
**错误信息**：`ConnectionError`, `TimeoutError`, `MootdxException: 连接服务器失败`

**可能原因**：
- 网络不稳定
- 默认服务器IP已失效

**解决方法**：
- 技能会自动尝试重连（最多3次）
- 手动在终端运行 `mootdx bestip -v` 更新服务器列表
- 从通达信官方客户端 `connect.cfg` 文件中提取最新IP，手动写入配置文件

### 2. 股票代码不存在或无数据
**错误信息**：`未找到代码 xxx 的行情数据`

**可能原因**：
- 输入代码错误（如少写一位）
- 该代码今日停牌或无行情数据

**解决方法**：
- 确认股票代码格式正确（6位数字）
- 检查是否为交易日交易时段

### 3. 财务数据获取失败
**错误信息**：`未找到 xxx 的资产负债表数据`

**可能原因**：
- 股票代码输入错误
- 该股票未披露对应报表

**解决方法**：
- 确认代码正确，并注意市场区分（60开头为上交所，00/30开头为深交所）

### 4. 本地数据读取路径错误
**错误信息**：`通达信路径无效或不存在`

**解决方法**：
- 提供正确的通达信安装目录路径，例如 `D:\new_tdx`
- 确保路径下包含 `vipdoc`、`T0002` 等数据文件夹

## 技能自动处理机制
- **自动重试**：所有网络请求均会尝试3次，每次间隔递增
- **自动优选服务器**：每次初始化客户端时会执行服务器测速
- **资源自动释放**：每次请求后自动关闭客户端连接
