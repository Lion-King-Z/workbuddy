## 信源缺失时的回退策略

| 情况 | 处理方式 |
|:---|:---|
| 需查公告但工具不支持直联 | 告知用户"需自行核实 + 提供路径" |
| 需行业数据但无付费账号 | 标注"免费口径/未见付费验证" |
| 结论仅来���一条来源 | 强制声明"单一信源，建议交叉验证" |
| 结论来自D级信源 | 降级输出"线索/推测，非结论" |
| 日期敏感数据（>30天） | 声明"数据截至日期，可能已变化" |
