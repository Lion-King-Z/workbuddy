# 常总-估值与逆向买入

## R — 原文

> 估值不是算命，是理解"市场共识的形成机制"。价格永远是市场对"未来预期"的折现。真正成熟的投资者，必须理解一句话：估值方法没有高低，只有是否"匹配阶段"。脱离企业发展阶段谈估值，等同于盲人摸象。暴跌敢买不是因为勇敢，而是因为看清价值；暴涨舍卖不是因为悲观，而是因为理解周期。
>
> — 常士杉，《伟大成长上市公司的终极估值大法——穿越周期的估值逻辑，而非数字游戏》 / 《暴跌敢买，暴涨舍卖——市场逆向思维的七大终极运用哲学》

---

## I — 方法论骨架

这个 Skill 做一件事：把"贵不贵、能不能买"从感觉变成流程。它不先算 PE，而是先问公司处于什么阶段；阶段决定用 PEG、PB 还是 PS；阶段和估值一起决定价格位置；位置和三重买入条件一起决定能否逆向买入。最后再用"价值想象力"做一次过滤，防止把故事当真、把真成长当垃圾。整个流程只有一条主线：阶段 → 估值工具 → 价格位置 → 买入条件 → 想象验证。

---

## A1 — 书中的应用

1. **成长期 PEG 错杀型**：某成长股市盈率 50 倍，利润增速 80%，PEG 约 0.6。股价因大盘暴跌或财报短期不及预期回撤 50% 以上，但营收增速仍 >30%、行业空间未变。属于"基本面未坏 + 情绪错杀"，落入三低后等待右侧确认。
2. **成熟期 PB 锚定型**：传统能源或银行类标的，利润增速低，但 PB 跌至 1 倍以下，行业现金流稳定。此时用 PEG 会失真，用 PB 看净资产安全边际，判断价格是否处于历史低位。
3. **爆发前 PS 斜率型**：AI/生物医药/软件平台公司尚未盈利，但收入连续多个季度加速，产品被验证。PE 为幻觉，PB 无效，用 PS 看收入增长的斜率，确认估值是否匹配产业爆发节奏。

## A2 — 触发场景

1. 用户问"这个公司现在估值贵不贵/能不能买"
2. 用户问"PEG 怎么看/现在多少倍算合理"
3. 用户问"暴跌了能不能抄底/三低条件满足了吗"
4. 用户问"这个标的处于高位还是低位"

语言信号：
- "这个公司估值高不高？"
- "PEG 怎么看？"
- "现在暴跌能不能买？"
- "三低买入条件满足了吗？"
- "是不是在高位？"
- "现在贵不贵？"

与相邻 skill 区分：
- 与 `常总-赛道稀缺卡位评估` 的区别：该 skill 判断赛道是不是时代主线、公司有没有稀缺卡位；本 skill 判断已经进入候选池后，当前价格是否低估、能否买入。
- 与 `常总-资金趋势识别` 的区别：该 skill 识别资金流入、放量、机构建仓等趋势信号；本 skill 用估值和逆向条件判断价格位置，只在最后一步参考技术/资金的右侧确认。
- 与 `常总-持有纪律与公司排雷` 的区别：该 skill 负责持仓中的风控、止损和排雷；本 skill 负责买入前的估值与逆向买入决策。

## E — 可执行步骤

当 skill 被激活后，按以下步骤执行：

### 步骤 1：估值阶段匹配

判断标的当前处于哪个阶段，从而选择估值语言。

- **成长期**：已有稳定利润，未来 3–5 年业绩爆发确定性高，行业空间巨大。典型：营收增速 >30%，EPS 持续提升，利润增速清晰。
- **成熟期**：利润增速放缓，行业格局稳定，现金流稳定。典型：银行、能源、公用事业、重资产制造业、资源类公司。
- **爆发前**：尚未盈利或微利，但产品被市场接受、收入处于加速斜率。典型：AI、软件平台、创新药、新能源前沿公司。
- **空想期**：只有故事、概念、PPT，没有稳定收入、没有商业闭环、没有可验证增长路径。

完成标准：给出一个明确的阶段标签（成长期 / 成熟期 / 爆发前 / 空想期），并列出判断依据。

判停条件：
- 若阶段判定为**空想期**，直接停止本 skill，输出"不可估、不可投"，并说明原因。
- 若数据不足无法判断阶段，向用户索要：营收增速、净利润增速、行业属性、收入趋势、是否盈利。

---

### 步骤 2：估值方法应用

根据阶段选择对应的估值工具，并给出高估/低估/中性结论。

- **成长期 → PEG 法**
  - 公式：PEG = PE / 净利润增速（一般用未来 1–2 年预期增速）
  - 判断：
    - PEG < 0.5：极致错杀（黄金坑）
    - 0.5–1：合理低估
    - 1–1.5：中性
    - > 1.5：高估
  - 重点关注：PEG < 0.7 纳入重点战机池
  - 输入：PE、未来净利润增速、行业空间
  - 输出：PEG 值、低估/中性/高估结论、是否进入重点战机池

- **成熟期 → PB 法**
  - 判断：
    - PB < 1：通常意味着极度悲观或系统性风险
    - PB 1–1.5：合理区间
    - PB > 2：往往需要周期、垄断或资源稀缺性支撑
  - 输入：PB、净资产、行业平均水平、历史 PB 中枢
  - 输出：PB 位置、安全边际判断

- **爆发前 → PS 法**
  - 核心不是现在赚不赚钱，而是：产品是否被接受、收入增长是否具备爆发性、商业模型是否可复制放大。
  - 判断：
    - 收入连续 2–3 个季度加速
    - 市场对其未来收入斜率给出溢价时，PS 高是合理的；若收入斜率走平或下降，PS 高即泡沫
  - 输入：收入增速、季度收入环比/同比、行业可比公司 PS
  - 输出：收入斜率判断、PS 是否合理

- **空想期 → 不可估**
  - 已在本步骤前的判停条件中处理，无需再估值。

判停条件：
- 若缺少关键估值数据（PE、增速、PB、PS、收入），向用户明确索要。
- 若估值结论为"高估"，仍需继续走完步骤 3–5，确认是否因为阶段错配或情绪错杀而出现机会；但必须在结论中标注"当前估值偏高，需等待错杀"。

---

### 步骤 3：价格位置判断

从六个维度综合判断"当前价格贵还是便宜"，不是只看股价，而是看市场位置。

1. **时间位置（产业周期）**：产业处于初创期、高速成长期、成熟期还是衰退期？价格高不代表位置高，关键看产业还有几层浪、未来空间还有多大。
2. **行业位置（竞争格局）**：标的是龙头、老二还是边缘公司？龙头即使 PE 高，利润持续增长后也会显得便宜；垃圾公司 PE 低，利润下降后反而越来越贵。
3. **市场位置（主场/客场）**：同一家公司在 A 股、港股、美股定价不同，要看所在市场的流动性、资金偏好和估值天花板。
4. **情绪位置（贪婪/恐惧）**：市场疯狂时风险大，恐慌时机会近。价格很多时候只是情绪的倒影。
5. **估值位置（PE/PEG/PB/PS）**：结合步骤 2 的估值工具，判断当前估值是否处于自身历史中枢低位或行业低位。
6. **技术位置（均线、成交量、趋势）**：辅助验证股价是否处于均线系统低位、成交量是否缩至地量、是否跌破关键支撑或形成底部结构。

完成标准：给出六维位置的综合判断，明确指出"山脚 / 山腰 / 山顶 / 无法判断"，并列出每个维度的关键证据。

判停条件：
- 若六个维度出现明显矛盾（例如估值低但情绪仍狂热），在输出中标注"位置判断存在矛盾，建议等待信号统一"。
- 若六维中有超过三个维度同时指向"山顶"，直接输出"当前价格处于高位区域，不建议逆向买入"。

---

### 步骤 4：逆向买入条件

只有当价格位置显示"处于低位或错杀区"，才进入本步骤。本步骤用"三低 + 暴跌敢买前提 + 右侧确认"判断能否买入。

#### 4.1 三低买入条件

- **低价**：股价相对自身历史估值中枢、行业平均水平显著偏低，或相对内在价值存在错配。本质是"好公司遭遇坏心情"，不是"坏公司垂死挣扎"。
- **低量**：底部区域成交量持续萎缩至地量，恐慌情绪释放殆尽，持股者惜售，持币者观望，空方力量衰竭。
- **低关注**：被主流资金遗忘、缺乏分析师覆盖、被媒体唱空、缺乏舆论热度。避免人声鼎沸的热门股。

完成标准：逐条检查三低是否满足，并给出证据。

判停条件：
- 若三低中任意一项不满足，输出"三低条件未完全满足，建议继续观察"，并指出缺失项。

#### 4.2 暴跌敢买前提

暴跌敢买的前提是：**基本面没有崩塌**。必须区分两种下跌：

- **逻辑破坏型下跌**：行业空间消失、核心竞争力瓦解、现金流恶化、增长逻辑断裂 → 永远不碰。
- **情绪错杀型下跌**：财报短期不及预期、行业恐慌、大盘暴跌、流动性冲击，但行业空间、核心竞争力、现金流、增长逻辑仍在 → 重点观察。

完成标准：明确判断下跌属于"逻辑破坏型"还是"情绪错杀型"，并给出证据。

判停条件：
- 若判断为逻辑破坏型，直接输出"不满足暴跌敢买前提，停止买入"，并建议转交 `常总-持有纪律与公司排雷` 进一步排雷。
- 若无法判断基本面是否死亡，向用户索要：最新财报、现金流、行业政策、核心竞争力变化。

#### 4.3 右侧确认

三低满足且是情绪错杀，仍不直接满仓买入。需要右侧确认信号：

- 突破年线或关键阻力位
- 出现放量信号
- 形成 W 底或类似底部结构
- 资金出现异动或机构建仓迹象

完成标准：列出右侧确认信号，并判断是否已出现。

判停条件：
- 若右侧确认信号未出现，输出"处于观察区，等待右侧确认"。
- 若右侧确认信号出现，输出"可以进入分批建仓流程"，并给出建议仓位（试探仓 20% → 确认加仓 → 突破重仓）。

---

### 步骤 5：价值想象验证

最后一步，防止把"故事"当"成长"，也防止把"真成长"当"垃圾"。

判断标准是：

- **真实产业支撑**：公司所在的产业趋势是否真实存在，不是纯概念炒作。
- **数据验证**：营收、用户、订单、产能、渗透率等关键指标是否有可验证的增长数据。
- **资金进入**：是否有机构、北向、产业资本等持续进入，而非散户跟风。

完成标准：明确回答"真想象力"还是"假想象力"，并给出证据。

判停条件：
- 若只有故事、没有收入、没有数据验证、没有资金持续进入 → 输出"假想象力，排除"。
- 若具备真实产业支撑、数据验证、资金进入 → 输出"真想象力，价值想象空间有效"。
- 若产业支撑强但数据验证不足，输出"想象空间存在，但需持续跟踪数据验证"。

---

### 最终输出格式

执行完毕后，统一输出：

1. **阶段与估值结论**：成长期/成熟期/爆发前，对应 PEG/PB/PS 值及高估/低估判断。
2. **价格位置**：六维位置综合判断（山脚/山腰/山顶）。
3. **逆向买入条件**：三低满足情况、暴跌敢买前提是否成立、右侧确认是否出现。
4. **价值想象验证**：真/假/待验证。
5. **综合操作建议**：
   - 可买入：给出分批建仓结构和关键止损位（跌破成本 -20% 无条件止损）。
   - 不可买入：明确说明原因和下一步观察条件。
6. **风险提示**：标注阶段错配、数据不足、情绪极端等风险。

## B — 边界

### 不要在以下情况使用此 skill

- 用户只想判断行业方向或赛道是否值得长期陪伴 → 转 `常总-赛道稀缺卡位评估`
- 用户想判断公司是否具备稀缺卡位或定价权 → 转 `常总-赛道稀缺卡位评估`
- 用户只想看资金趋势、放量、机构建仓 → 转 `常总-资金趋势识别`
- 用户已持仓，需要止损、风控、排雷 → 转 `常总-持有纪律与公司排雷`
- 标的是概念股、PPT 公司、没有收入、没有商业闭环 → 本 skill 直接输出"不可估"

### 作者在书中警告的失败模式

- 只看 PE，不看阶段：对成长期用 PB、对成熟期用 PEG 都会失真。
- 只看价格，不看位置：涨了怕高、跌了怕低，容易被情绪牵着走。
- 把暴跌当机会，但买的是基本面崩塌的公司：这是逆势亏损，不是逆向投资。
- 盲目抄底，没有右侧确认：底部区域可以观察，但重仓需要放量或结构确认。
- 把想象力当确定性：只有故事、没有收入和资金验证的公司，容易价值归零。

### 作者的盲点 / 时代局限

- 估值阈值（如 PEG < 0.7、PB 1–1.5）来自 A 股和美股部分成长股经验，在流动性极度宽松或紧缩的环境中可能整体上移或下移。
- 对尚未盈利的科技公司，PS 法高度依赖市场风格，牛市中容易给出过高溢价，熊市中溢价可能快速坍塌。
- 三低买入法强调耐心和逆人性，对短线资金或必须快速出业绩的账户可能不适合。
- 部分判断（如"情绪位置""市场位置"）带有主观性，需要结合数据和市场温度共同判断。

### 容易混淆的邻近方法论

- 与价值投资：本 skill 也看 PB、安全边际，但更强调阶段匹配和右侧确认，不是纯深度价值长期持有。
- 与趋势投资：本 skill 最后一步参考右侧确认，但核心逻辑是"低估 + 错杀 + 价值想象"，不是追涨杀跌。
- 与纯量化估值：本 skill 允许主观判断（情绪、市场位置、价值想象），不是只看财务指标。
