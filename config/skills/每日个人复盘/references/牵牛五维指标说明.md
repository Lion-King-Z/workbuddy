# 牵牛五维指标说明

## 指标来源

朋友方案中使用的5个技术指标，用于量化持仓标的的技术健康状态。本skill默认继承该框架，权重可调。

---

## D1 趋势（今日收涨）

| 项目 | 说明 |
|:---|:---|
| 含义 | 标的当日是否收涨 |
| 通过标准 | 当日涨跌幅 > 0 |
| 未通过含义 | 短期趋势走弱，当日出现负反馈 |
| 权重 | 20% |
| 数据源 | tdx-connector `chg_percent` |

---

## D2 中线（站稳BBI）

| 项目 | 说明 |
|:---|:---|
| 含义 | 价格是否站在BBI（多空指数）上方 |
| 通过标准 | 收盘价 > BBI |
| 未通过含义 | 中线趋势破位，可能进入调整 |
| 权重 | 25% |
| 数据源 | tdx-connector 计算指标 |

**BBI计算公式**：
```
BBI = (3日MA + 6日MA + 12日MA + 24日MA) / 4
```

---

## D3 筹码（非放量杀跌）

| 项目 | 说明 |
|:---|:---|
| 含义 | 下跌时是否伴随恐慌性放量 |
| 通过标准 | 当日成交量 < 5日均量 × 1.2（或下跌时未显著放量） |
| 未通过含义 | 放量杀跌，筹码松动，可能存在机构出货 |
| 权重 | 20% |
| 数据源 | tdx-connector `volume` + 5日平均 |

---

## D4 拐点（KDJ未死叉）

| 项目 | 说明 |
|:---|:---|
| 含义 | KDJ指标是否维持多头状态 |
| 通过标准 | K值 > D值（未形成死叉） |
| 未通过含义 | KDJ死叉，短期动能转弱 |
| 权重 | 15% |
| 数据源 | tdx-connector 计算指标 |

**默认参数**：KDJ(9,3,3)

---

## D5 斜率（多头排列）

| 项目 | 说明 |
|:---|:---|
| 含义 | 短期均线是否呈现多头排列 |
| 通过标准 | MA5 > MA10 > MA20 |
| 未通过含义 | 多头排列破坏，趋势结构转弱 |
| 权重 | 20% |
| 数据源 | tdx-connector 计算指标 |

---

## 加权评分算法

```
加权评分 = D1×0.20 + D2×0.25 + D3×0.20 + D4×0.15 + D5×0.20

其中 Dn 为通过=1，未通过=0
```

| 评分区间 | 状态 |
|:---:|:---|
| 4.0-5.0 | 优秀 |
| 3.0-4.0 | 一般 |
| 2.0-3.0 | 偏弱 |
| 0-2.0 | 危险 |

---

## 阈值说明

各维度的健康阈值来自朋友方案：

| 维度 | 通过数/总数阈值 | 含义 |
|:---|:---:|:---|
| D1 | ≥50% | 半数以上标的收涨 |
| D2 | ≥70% | 大多数标的中线趋势完好 |
| D3 | ≥90% | 极少出现放量杀跌 |
| D4 | ≥70% | 大多数标KDJ未死叉 |
| D5 | ≥80% | 大多数标多头排列 |

---

## 注意事项

1. 技术指标只是复盘的辅助工具，不构成买卖信号
2. 单一标的D1-D5低分需要结合基本面和估值综合判断
3. 高位超买标的（KDJ超买+布林上轨）即使五维通过，也需要警惕过热
