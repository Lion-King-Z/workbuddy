# 每日窄门筛股 Skill v3.1 🆕

**定位**：从第一性原理出发——**投资就是找「供给卡死 + 市场还没发现」的交集**。证据累积（E）不应该是独立维度——等到证据累积够了，股价早涨完了。E只是催化剂链上的一环验证，不是评分维度。

**v3.1 核心升级**：+催化剂兑现后操作流程（三步决策+涨幅比）。催化剂兑现后不是"拿住等下一个"——重新测算当前价vs三层目标价，给涨幅比，让数据说话。
**v3.0**：四因子(B/E/C/G) → 三因子(B/G/催化剂)。C降级为质量折扣乘数。E消亡。🏆黄金赔率检测。

---

## 第一性原理

### 赚钱的底层公式

```
赚钱 = 供给卡死的标的 × 市场还没发现 × 有催化剂让市场发现
     = B(瓶颈强度) × G(预期差) × 催化剂(时机)
```

三个维度必须同时成立：
- B高但G低 = 好公司但已经涨完了（如现在的电子布）
- G高但B低 = 便宜但供给不刚性，随时被替代
- BG都高但无催化剂 = 遥遥无期的价值，资金效率低

### 三因子独立定义

| 维度 | 衡量什么 | 权重 | 高意味着 | 低意味着 |
|:---|:---|:---:|:---|:---|
| **瓶颈强度 B** | 供给是否有物理/政策/技术硬约束 | 30% | 稀缺会持续，卡位不可替代 | 供给随时可缓解 |
| **预期差 G** | 市场定价 vs 真实紧缺的背离程度 | 40% | 市场还没发现，上行空间大 | 已经充分定价，利好出尽 |
| **催化剂链** | 未来1-6个月会有什么事让市场重新定价 | 30% | 有明确触发事件，钱快到了 | 遥遥无期，资金效率低 |

G权重最高（40%）——因为**预期差 = 赚钱空间**。B是地基（30%），催化剂是扳机（30%）。

### 确定性C的新定位：质量折扣乘数（不是独立维度）

```
综合分 = (B × 30% + G × 40% + 催化剂 × 30%) × C乘数
```

| 确定性等级 | C分范围 | C乘数 | 含义 |
|:---|:---:|:---:|:---|
| 高确定 | C ≥ 8 | **×1.0** | 信源顶级+关键假设全部验证，不打折 |
| 中高确定 | C 6-7 | **×0.9** | 信源良好+主要假设已验证，轻微折扣 |
| 中等确定 | C 4-5 | **×0.75** | 部分验证有不确定性，显著折扣 |
| 低确定 | C 2-3 | **×0.50** | 多数假设待验证，大幅折扣 |
| 极不确定 | C ≤ 1 | **×0.30** | 基本全在假设阶段，极度折扣 |

**C的判据不变**（沿用v2.x的C1信源质量+C2关键假设验证率的平均值）。

**核心逻辑**：C不参与维度权重竞争。B和G决定「机会有多大」，C决定「你敢放多少钱」。高C=满血出击，低C=打折防御。

### 🏆 「确定性中高但预期差大」—— 黄金赔率标的

这是三因子框架下最适合重仓的目标类型：

| 特征 | 标准 | 为什么值钱 |
|:---|:---|:---|
| 供给卡死确定性中高 | C ≥ 4 | 瓶颈真实存在，不会血本无归 |
| 预期差大 | G ≥ 7 | 市场没发现，上行空间充足 |
| 有催化剂 | 催化剂 ≥ 5 | 近期有事件让市场重新定价 |

**怎么找到这类标的？**
1. 扫描浑水调研高频提及的品种 → 这些是B和C有基础的
2. 逐一检查G（WebSearch热度 + 券商覆盖数 + PEG水位） → 筛出市场关注度低的
3. 验证催化剂链（近期有验证事件吗？） → 确认有扳机
4. 输出「黄金赔率」标签 → C≥4 + G≥7 + 催化剂≥5

**反例**：WF6现在B=9（强瓶颈）、C=7（高确定），但G已降到5（机构开始覆盖），催化剂大多已兑现——这是「高确定低预期差」，不是黄金赔率，是鱼尾。

### 仓位 = 赔率 × 胜率 × 确定度

```
赔率 = B × G（供给刚性 × 市场没发现）
胜率 = 催化剂链兑现概率
确定度 = C乘数（信源质量+假设验证率的折扣）

仓位系数 = (综合分 ÷ 10) × 100%
```

不同得分区间对应不同仓位：

| 综合分 | 仓位系数 | 仓位建议 | 思维框架 |
|:---:|:---:|:---|:---|
| ≥8.0 | 80-100% | 重仓（仓位上限×100%） | 三因子全强共振 |
| 6.5-7.9 | 65-80% | 标准（仓位上限×70%） | 强两因子+中一因子 |
| 5.0-6.4 | 50-65% | 试探（仓位上限×30%） | 有一端略逊 |
| 3.5-4.9 | 35-50% | 观察（不持仓） | 值得跟踪等突破 |
| <3.5 | — | 放弃追踪 | 不浪费注意力 |

**特别规则**：C≥4 + G≥7（黄金赔率）→ 仓位系数自动+10%，因为确定性中高意味着底牌不差，预期差大意味着上方空间足。这是三因子框架下最值得出手的组合。

### 快速3维评分模式（保留，星大降维）

当信号量大（>15条）或需要快速初筛时，先用3维快速模式过滤：

| 维度 | 定义 | 快速判据 | 评分规则 |
|:---|:---|:---|:---|
| **重要性** | 下游刚需，不可替代 | 该材料/环节是否是AI/半导体制造必须的？ | 5=不可替代+刚需 / 3=有替代但成本高 / 1=可替代 |
| **紧缺度** | 供给壁垒 + 扩产周期 | 扩产周期？国产化率？认证壁垒？ | 5=扩产>18月+国产<20% / 3=扩产12-18月 / 1=扩产<12月 |
| **空间潜力** | 还没涨 + 有催化 | 估值水位？催化剂时点？市场关注度？ | 5=没涨+强催化+大空间 / 3=部分涨 / 1=涨透 |

```
快速3维总分 ≥ 12 → 🔴 深水区 → 进入完整三因子评分(B/G/催化剂)
快速3维总分 9-11 → 🟡 观察区 → 记录，等关键验证点
快速3维总分 < 9 → ⚪ 过滤 → 不占注意力
```

## 工作流：三因子加权评分

### Step 0：信号汇总与去重

```
1. 读取今日三条产出线的全部信号
2. 按行业/环节去重合并
3. 标注来源：紧缺度日报 | 电话会 | 推送 | 聚宝盆缓存
4. 形成候选信号池
```

🔴 **CHECKPOINT**：确认三条产出线至少一条有数据。若均空→用昨日窄门筛股结果做延续性分析。信号<3条→正常输出不凑数。

### Step 1：瓶颈强度 B 评分（0-10分，权重30%）

从产业链瓶颈投研skill提取，5个维度，每维 0-2 分：

| 维度 | 0分 | 1分 | 2分 |
|:---|:---|:---|:---|
| **供给刚性** | 扩产<12个月 | 扩产12-18个月 | 扩产>18个月，有认证/建厂硬约束 |
| **量化缺口** | 无数字，纯定性描述 | 有数字但未交叉验证 | 有量化缺口且被多源确认 |
| **不可替代性** | 替代品充足且价格正常 | 替代品存在但也在涨价 | 无替代品，或替代品被同一瓶颈卡住 |
| **价格传导** | 未涨价或涨价被下游拒绝 | 涨价已被下游部分接受 | 涨价全链条传导，下游确认接受 |
| **需求端刚性** | 需求可推迟/可替代 | 需求有粘性但短期可调节 | 需求无弹性（停产=断供下游） |

**B = 5维度加总（0-10分）**

供给刚性分类映射（来自产业链瓶颈投研skill v3.2）：
- 产能约束型 → 重点看供给刚性+量化缺口维度
- 认证锁定型 → 重点看不可替代性+价格传导维度
- 地缘切割型 → 重点看不可替代性+需求端刚性维度

🔴 **CHECKPOINT**：B≥8或B≤2 → 标注极端值。B≤2 → 不论其他维度多高，综合分上限锁定5.0。

### Step 2：预期差 G 评分（0-10分，权重40%）

衡量"市场定价 vs 真实紧缺的背离程度"。**G是赚钱空间的核心衡量**。

| 子维度 | 0-0.5分 | 1-1.5分 | 2-2.5分 |
|:---|:---|:---|:---|
| **市场关注度** | 微博热搜/各大财经头条在推，WebSearch>20条 | 仅浑水调研+少数机构提及，WebSearch 5-20条 | 几乎无公开讨论，WebSearch<5条 |
| **估值水位** | PEG>2.0或PE高于行业中枢50%+ | PEG 1.5-2.0或PE高于中枢20-50% | PEG<1.0或PE低于中枢，或无人覆盖 |
| **催化剂时点** | 催化剂已兑现超过1个月 | 1个月内催化剂即将兑现 | 3个月以上远期催化剂 |
| **持仓拥挤度** | 多家头部机构重仓，券商覆盖≥5家 | 开始有1-2家机构覆盖 | 几乎无机构覆盖，券商0-1家 |

**G = 四子维度加总（上限10分）**

**G判据获取路径**：
- 市场关注度 → WebSearch搜索量 + 雪球热帖数
- 估值水位 → tdx-connector PE数据 + 行业中枢PE
- 催化剂时点 → 浑水跟踪表"供给事件"列
- 持仓拥挤度 → 券商覆盖家数（WebSearch"标的简称 + 研报"）

🔴 **CHECKPOINT**：G≥8 → 极高预期差。但需配合C验证——C≤3且G≥8可能是「真的没发生」而非「市场没发现」。标注「高G低C，需区分未定价vs未发生」。

### Step 2.5：黄金赔率检测（🏆 新增）

**C ≥ 4 且 G ≥ 7** → 标注 **🏆 黄金赔率**。

| 检测项 | 标准 | 含义 |
|:---|:---|:---|
| 供给真的卡死了吗？ | C ≥ 4（确定性中高） | 不是瞎猜，是确认真实的瓶颈 |
| 市场还没发现吗？ | G ≥ 7（预期差大） | 上方空间充裕 |
| 有扳机吗？ | 催化剂 ≥ 5 | 近期有事件让市场发现 |

🏆 黄金赔率标的的仓位系数自动 **+10%**（在Step 5中体现）。

### Step 3：催化剂链评分（0-10分，权重30%）

衡量"未来1-6个月有什么事件会让市场重新定价"。

**催化剂链三层结构**（来自产业链瓶颈投研skill第五章）：

| 催化剂层 | 时间窗口 | 评分贡献 | 典型事件 |
|:---|:---|:---:|:---|
| 🔴 **近期触发器** | 1-2月 | **0-5分** | 客户验证公告/季报预增/涨价函/产能利用率数据 |
| 🟠 **中期验证器** | 3-4月 | **0-3分** | 新产能投产/在手订单兑现/财报验证 |
| 🟡 **远期格局兑现** | 5-6月 | **0-2分** | 国产替代政策落地/市占率跃升/行业拐点 |

**评分规则**：
- 近期有≥2个可验证催化事件 → 5分
- 近期有1个可验证催化事件 → 3分
- 近期无但有中期可验证事件 → 2分
- 仅有远期事件 → 1分
- 无任何可验证催化事件 → 0分

**催化剂链与证据累积的关系**：
E不再是独立维度，而是融入催化剂链验证——每个催化剂标注"验证状态"：
- ✅ 已验证 → 该层催化剂已被市场部分定价，G可能已缩小
- 🔄 进行中 → 催化正在发生，是入场窗口
- ⏳ 待触发 → 未来事件，市场未定价

**这是E的信息增量逻辑的新家**：不再累积"同一信号被提了多少次"，而是追踪"催化剂链上的事件是否如期兑现"。

🔴 **CHECKPOINT**：催化剂=0 → 无论B/G多高，综合分上限锁定5.5。没有扳机的枪等于废铁。

### Step 3.5：确定性 C 判定（质量折扣乘数）

沿用v2.x的C评分体系，但不再参与维度权重：

**C1 — 信源质量（0-5分）**：

| 信源 | C1 |
|:---|:---:|
| 浑水调研原文（P0）+ 法定公告（A级）交叉验证 | 5 |
| 浑水调研原文（P0）+ 行业数据（B级）交叉验证 | 4 |
| 浑水调研原文（P0）单一来源 | 3 |
| WebSearch公开数据 + 行业数据 | 2 |
| 单一WebSearch新闻 | 1 |

**C2 — 关键假设验证率（0-5分）**：

| 验证率 | C2 | 含义 |
|:---:|:---:|:---|
| ≥80% | 5 | 高度确定 |
| 60-80% | 4 | 主要假设已验证 |
| 40-60% | 3 | 部分验证 |
| 20-40% | 2 | 多数待验证 |
| <20% | 1 | 基本全假设 |

**C = (C1 + C2) / 2**，映射到乘数：≥8→×1.0 / 6-7→×0.9 / 4-5→×0.75 / 2-3→×0.50 / ≤1→×0.30

🔴 **CHECKPOINT**：C≤2 + G≥8 → 🔴「高预期差可能是假象」风险标注。C≤1 → 综合分上限再降10%（低确定高赔率可以小注，但不能重仓）。

### Step 4：综合评分计算

```
综合分 = (B × 0.30 + G × 0.40 + 催化剂 × 0.30) × C乘数

约束规则：
- B≤2 → 综合分 ≤ 5.0（瓶颈弱=基础不牢）
- 催化剂=0 → 综合分 ≤ 5.5（无扳机=废铁）
- C≤1 → 综合分上限再降10%（极不确定）
- C≥4 + G≥7 → 仓位系数+10%（黄金赔率溢价）
```

### Step 4.5：预期差分层定价（G层级映射，保留v2.3机制）

| G层级 | G分范围 | 市场认知水平 | 仓位建议 | 
|:---|:---:|:---|:---|
| 🔴 **深度忽视层** | 9-10 | 几乎无人知晓 | 试探级(30%)——无共识保护 |
| 🟠 **初步认知层** | 6-8 | 少数人关注 | 标准级(70%)——市场开始验证 |
| 🟡 **扩散定价层** | 3-5 | 机构开始覆盖 | 标准级但不追加——鱼身吃完 |
| ⚪ **充分定价层** | 1-2 | 热门拥挤 | 退出信号 |

三层目标价：

```
T1（认知启动价）= 一致预期EPS(N+1) × 行业中枢PE × (1 + B溢价)
T2（紧缺确认价）= T1 × (1 + 催化剂链近期兑现溢价)
T3（充分定价价）= T2 × (1 + 催化剂链全兑现溢价)

T3 - 当前价 = 总理论上行空间
```

### Step 4.6：催化剂兑现后操作流程（v3.1 新增 — 关键修正）

**核心原则**：催化剂兑现后不是"拿住等下一个"——是**重新算一遍，该走走**。中期催化剂还要3-4个月，时间成本和机会成本必须在操作决策中量化为数字。

#### 流程

```
催化剂兑现后 → 三步决策：

第一步：重新测算当前价 vs 三层目标价
  当前价 vs T1（认知启动价）：已到/未到
  当前价 vs T2（紧缺确认价）：已到/未到   
  当前价 vs T3（充分定价价）：已到/未到

第二步：根据兑现后的位置决定操作
  ┌─ 当前价 ≥ T2（核心预期已兑现）
  │   → 建议止盈。计算数据：
  │   「本轮收益 = (当前价-买入价)/买入价 = XX%」
  │   「下一催化触发后T3预期 = XX元，较当前价涨幅 = XX%」
  │   「等待成本 = 3-4个月时间+机会成本 ≈ XX%/年化」
  │   「涨幅比 = 等待收益% / 当前止盈收益%」
  │   → 涨幅比>2:1才值得等，否则建议走
  │
  ├─ 当前价 < T2，但趋势线全部破位
  │   → 建议减仓至试探级(≤30%上限)。给出：
  │   「趋势面：均线/支撑/量能全部恶化」
  │   「基本面：B未变，远期G还在」
  │   「操作建议：减仓止盈部分，留底仓等下一催化」
  │   → 等下一催化剂触发+趋势修复再补回
  │
  └─ 当前价 < T2，趋势完好
      → 继续持有标准仓位，等下一催化剂
      → 但下一催化 >3个月 → 标注"等待期"，定期复核趋势

第三步：如果决定等下一催化剂
  📅 下一催化剂预计：YYYY.MM（约N个月后）
  💰 当前止盈收益：XX%
  🎯 等下一催化触发后T2预期：XX元（较当前+XX%）
  ⏱️ 等待成本：N个月×年化机会成本XX% = XX%
  📊 涨幅比：等待收益/当前止盈 = XX% / XX% = X.X:1
  
  建议规则（⚠️ 非强制一票否决）：
  - 涨幅比 > 3:1 → 强烈建议等（下一催化空间远大于当前）
  - 涨幅比 2-3:1 → 可以等等，但建议设紧止损
  - 涨幅比 1-2:1 → 倾向止盈，等催化触发再买回来
  - 涨幅比 < 1:1 → 强烈建议先走，催化触发后再参与也不迟
```

**WF6案例示范**（为什么B和G不是天然对立——这是反面案例但操作逻辑上是正面）：

```
当前现状（假设）：
  当前价 已接近T2（紧缺确认被市场定价）
  下一催化剂：Q3季报验证（约3个月后）
  
  第一步：当前价约=XX元，T2=XX元 → 当前价 ≥ T2
  第二步：核心预期已兑现 → 进入止盈评估
  第三步：
    💰 本轮收益 = 已赚XX%
    🎯 Q3季报后T3预期 = XX元 → 距当前+10%
    ⏱️ 等待成本 = 3个月机会成本（该涨的都涨了）
    📊 涨幅比 = 10% / XX% = 偏低
    → 建议：兑现核心收益，等Q3季报催化剂触发后重新评估
```

> **关键区分**：催化剂兑现≠基本面变差。B（瓶颈强度）没变，但市场已经发现并定价了这个瓶颈——G（预期差）缩小了。**这不是让你平仓放弃，是让你兑现收益后等下一轮催化再参与。** 鱼身吃完了，把筷子放下等下一盘，别拿筷子戳鱼刺。

### Step 5：仓位建议

**仓位上限** = 景气度skill 对应阶段上限

**仓位系数** = 综合分映射：

| 综合分 | 仓位系数 | 实际仓位 |
|:---:|:---:|:---|
| ≥8.0 | 80-100% | 上限×100% |
| 6.5-7.9 | 65-80% | 上限×70% |
| 5.0-6.4 | 50-65% | 上限×30% |
| 3.5-4.9 | 35-50% | 观察，不持仓 |
| <3.5 | — | 放弃追踪 |

**🏆黄金赔率加成**：C≥4+G≥7 → 仓位系数+10%

**止损线**：
```
止损比例 = 标准止损(15-20%) × (1 - C/10)
```
C越低止损越紧——低确定性的东西，错了要跑得快。

**风险收益比速算**：
```
上行空间 = (T3 - 当前价) / 当前价
下行风险 = 止损比例
风险收益比 = 上行空间 / 下行风险

≥ 3:1 → 高赔率，重仓
2-3:1 → 合理，标准
1-2:1 → 勉强，试探
< 1:1 → 不入场
```

🛑 **STOP**：持仓标的综合分<3.5 → 必须输出「建议平仓」。

## 核心原则

| # | 原则 | 含义 |
|:---|:---|:---|
| 1 | **供给卡死优先** | B是地基，B≤2的东西再好也上限锁定 |
| 2 | **预期差=赚钱空间** | G权重最高(40%)——市场没发现=钱没赚到 |
| 3 | **无扳机=废铁** | 催化剂=0的东西不给重仓，资金效率太低 |
| 4 | **确定性是乘数不是维度** | C不参与维度竞争——它决定你敢不敢重仓，不是机会大不大 |
| 5 | **黄金赔率>一切** | C≥4+G≥7是唯一自动加仓的信号——底牌不差+上方空间大 |
| 6 | **证据累积已死** | E不是独立维度。证据只是催化剂链的验证状态，不是评分项 |
| 7 | **G层级约束仓位** | 深度忽视层不给重仓（无共识保护），充分定价层只出不进 |
