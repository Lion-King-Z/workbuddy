# 每日窄门筛股 Skill v4.2

**定位**：买入信号**总调度**。不自己硬算——专业的事情交给专业的Skill做。

**v4.2 核心升级**：
1. 🆕 **实时行情验证** — Step 0.5通过 tdx-connector 获取最新股价，验证PE估值
2. 🆕 **事件→产业链前瞻映射** — Step 0.6 大事件（美光财报等）自动推导受益标的纳入候选池
3. 🆕 **涨幅风险量化** — Step 3e 近N日涨幅检测 + "是否已反映预期"判断
4. 🆕 **前瞻PE调整** — Step 3f 新事件修正N+1E盈利预期，PE不只盯着过去财报
5. 🆕 **催化剂新鲜度衰减** — 同催化剂今天 vs 3天前权重不同
6. 🆕 **板块联动提醒** — 事件→上下游传导链自动提示
7. 📊 **表格重构** — 精简为：排名/标的/综合分/关键信号/评级/风险提示/PE(TTM)

---

## 架构：五层委派+行情验证

```
信号池（窄门筛股收集）
    │
    ├─→ 实时行情采集 ──────────→ 最新股价 + 近N日涨幅
    ├─→ 事件→产业链映射 ──────→ 大事件受益标的自动入池
    │
    ├─→ 产业链瓶颈投研skill  ──→ B(瓶颈强度) + 供给刚性类型 + 紧缺信号
    ├─→ 景气度skill          ──→ 产业阶段(0-6级) + 材料子生命周期
    └─→ pe-valuation-analyst ──→ G(预期差) + PE估值 + 三层目标价 + PE陷阱
    │
    ↓
窄门筛股汇总：涨幅风险检测 + 前瞻PE调整 → 三因子评分 → 排序 → 买入建议
```

窄门筛股不重复造轮子——每个维度的评分由最懂它的Skill出。

## 工作流

### Step 0：信号收集（窄门筛股自己做）

1. 从三来源提取所有被提及的品种/标的
2. 去重合并，形成候选池
3. 标注来源和初步紧缺信号强度
4. 🔴 当日无产出→用昨日结果延续，标注"无新增信号"

### Step 0.5：🆕 实时行情采集（窄门筛股自己做）

对候选池中每个有股票代码的标的，通过 **tdx-connector（通达信MCP）** 获取实时行情：

```
tdx_quotes(code=股票代码)
→ 提取：最新价 / 涨跌幅 / 近5日涨幅 / 近20日涨幅 / 近60日涨幅
```

若tdx-connector不可用，降级为 **akshare-stock** 或 **westock-data**。

**产出**：每个标的的 `当前价` + `近5日涨跌幅` + `近20日涨跌幅` + `近60日涨跌幅`

### Step 0.6：🆕 事件驱动产业链前瞻映射（窄门筛股自己做）

**核心逻辑**：重大事件（如美光财报超预期）→ 产业链上下游受益标的 → 即使这些标的尚未出现在今日推送中，也应纳入候选池。

**映射规则**：

| 事件类型 | 受益方向 | 示例标的 |
|:---|:---|:---|
| 存储原厂财报超预期 | 封测/材料/设备 | 长电科技/雅克科技/中船特气 |
| 英伟达GPU放量 | 光模块/PCB/散热 | 新易盛/沪电股份/高澜股份 |
| AI资本开支上调 | 服务器/柴发/液冷 | 工业富联/潍柴重机/英维克 |
| 晶圆厂扩产 | 硅片/特气/靶材 | 有研硅/中船特气/江丰电子 |
| 封装技术突破(CoWoS等) | 封装基板/ABF/玻璃基板 | 深南电路/生益科技 |
| 终端产品发布(AI眼镜等) | 光学/显示/MCU | 水晶光电/韦尔股份 |

**操作**：
1. 扫描今日信息推送中的S级/A级信号
2. 对每个S/A级信号按上表推导受益方向
3. 将推导出的标的追加到候选池（标注来源为"事件映射"）
4. 事件映射标的也走完整委派评估流程

**示例**：今日美光财报→存储材料受益→追加雅克科技(HBM前驱体)、中船特气(WF6)、长电科技(封测)进入候选池，即使它们今日推送未直接提及。

### Step 1：B 瓶颈强度 → 委派给「产业链瓶颈投研skill」

对每个候选品种，使用产业链瓶颈投研skill的以下模块：

**1a. 供给刚性分类**（Skill Step 1.5）
- 判定类型：产能约束型 / 认证锁定型 / 地缘切割型
- 分类决定后续催化剂路径

**1b. 四维度深度评分**（Skill Step 2）
- 不可替代性(30%) + 供应弹性(30%) + 集中度(20%) + 认知偏差(20%)
- 使用供给刚性类型对应的调整后权重
- 输出：加权总分 → 映射到B(0-10)

**1c. 紧缺六维信号检测**（Skill 第三章）
- 价格/产能/客户行为/库存/投资/跨市场验证
- 确认信号数 → 辅助B验证

**B赋值规则**：

| 四维加权总分 | B分数 |
|:---:|:---:|
| ≥4.0 (🔴Red alert) | 8-10 |
| 3.5-3.9 (🟠Orange) | 6-7 |
| 3.0-3.4 (🟡Yellow) | 4-5 |
| 2.0-2.9 | 2-3 |
| <2.0 | 0-1 |

⚠️ B≤2 → 标注「供给刚性不足」，排序靠后，不推荐但不隐藏。

### Step 2：产业阶段 → 委派给「景气度skill」

对每个候选品种，使用景气度skill：

**2a. 判定产业阶段**（Skill Step 1）
- 通用7级：0级(技术验证)→6级(价值陷阱)
- 半导体材料走子生命周期：①验证→②导入→③小批量→④国产替代

**2b. 阶段=赔率定位**
- 0.5级→3.5级 = 黄金投资区间（一年十倍摇篮）
- 3.5级之后 = 吃业绩兑现的钱，弹性降低
- 5级+ = 弹性释放，赔率下降

**阶段映射到催化剂链权重调整**：

| 阶段 | 对催化剂的含义 |
|:---|:---|
| 1-2级（导入期） | 近期催化剂密集，权重上调 |
| 3级（产能爬坡） | 中期催化剂关键 |
| 4级+（业绩释放） | 催化剂已大量兑现，远期格局为主 |

### Step 3：G 预期差 + PE估值 → 委派给「pe-valuation-analyst」

对每个候选品种，使用pe-valuation-analyst：

**3a. PE陷阱检测**（Skill Step 0.6）
- 三大陷阱对号入座：泡泡玛特型/美团型/铜矿型
- 陷阱≠淘汰，陷阱=风险地图

**3b. 多档PE评估**（Skill Step 5）
- 静态PE / TTM PE / N+1E / N+2E
- PEG校验
- 增速持续性检验

**3c. 三层目标价**（Skill Step 6b/7）
- T1（认知启动价）= 基准净利 × 中枢PE × (1+B溢价)
- T2（紧缺确认价）= T1 × (1+近期催化溢价)
- T3（充分定价价）= T2 × (1+全催化溢价)
- 上行空间 = (T3-当前价)/当前价

**3d. G预期差评分**（从PE Skill的认知偏差+估值水位推算）

| G子维度 | 数据来源 |
|:---|:---|
| 市场关注度 | PE Skill的市场关注度判断 + WebSearch |
| 估值水位 | PE Skill的多档PE + PEG + 行业百分位 |
| 催化剂时点 | PE Skill Step 3催化剂追踪仪表盘 |
| 持仓拥挤度 | PE Skill的券商覆盖数 |

### Step 3e：🆕 涨幅风险检测（窄门筛股自己做）

基于 Step 0.5 获取的实时行情，对每个标的检测涨幅风险：

**检测维度**：

| 检测项 | 计算方式 | 风险信号 |
|:---|:---|:---|
| 短期冲高 | 近5日涨跌幅 | >20%→🔴高风险 / 10-20%→🟡注意 / <10%→🟢正常 |
| 中期累积 | 近20日涨跌幅 | >40%→🔴高风险 / 20-40%→🟡注意 / <20%→🟢正常 |
| 上行空间vs涨幅 | 预期上行空间 / 近20日涨幅 | <1.5x→涨幅已充分反映 / 1.5-3x→部分反映 / >3x→仍有空间 |
| 是否已反映预期 | 当前价 vs T1目标价 | 已超T1→预期已部分定价 / 逼近T2→接近充分定价 / <T1→预期未反映 |

**风险标注规则**：

| 条件 | 标注 |
|:---|:---|
| 近5日>20% 或 近20日>40% | ⚠️ 短期涨幅过大，追高风险 |
| 当前价已超T1 | ⚠️ 预期已部分反映，安全边际收窄 |
| 当前价逼近T2 | ⚠️ 预期接近充分定价，上行空间有限 |
| 上行空间/近20日涨幅 <1.5x | ⚠️ 涨幅已充分反映预期 |

### Step 3f：🆕 前瞻PE调整（窄门筛股自己做）

**核心逻辑**：PE Skill基于历史财报计算PE，但今日重大事件（如美光财报超预期）意味着未来盈利预期上修，TTM PE可能虚高。需要做前瞻性调整。

**调整规则**：

1. **事件→盈利预期修正**：
   - 若今日有产业龙头财报超预期（如美光），该链条上游标的的N+1E预期收入至少上调10-20%
   - 反映到前瞻PE = 当前市值 / (原N+1E × 1.1~1.2)

2. **涨价传导修正**：
   - 若产品已在涨价通道（如MLCC村田7/1提价），N+1E天然上修
   - 将涨价幅度折算进N+1E

3. **输出**：在PE评估中同时展示"TTM PE(历史)"和"前瞻PE(事件调整后)"，标注调整依据

### Step 4：C 确定性 → 汇总三个Skill的信源

从三个Skill的执行结果中提取信源等级：

| 信源组合 | C1 |
|:---|:---:|
| 产业链瓶颈skill T1 + PE skill [T1]双验证 | 5 |
| 产业链瓶颈skill T1 + PE skill [T2] | 4 |
| 单一T1来源 | 3 |
| T2/T3混合 | 2 |
| 单一WebSearch | 1 |

C2关键假设验证率 = 产业链瓶颈skill中量化缺口验证状态 + PE skill中增速持续性确认状态，取平均。

C = (C1+C2)/2 → 映射乘数：≥8→×1.0 / 6-7→×0.9 / 4-5→×0.75 / 2-3→×0.50 / ≤1→×0.30

### Step 5：综合评分

```
综合分 = (B × 0.30 + G × 0.40 + 催化剂 × 0.30) × C乘数

🏆 黄金赔率：C≥4 + G≥7 → 综合分+0.5
```

**催化剂链评分**：从产业链瓶颈skill第五章（催化剂链三层结构）+ PE Skill Step 3（催化追踪仪表盘）提取。

| 催化剂来源 | 提取内容 |
|:---|:---|
| 产业链瓶颈skill 第五章 | 近/中/远三层催化剂事件+定价升级路径 |
| PE Skill Step 3 | 催化事件+类型(A/B/C)+状态+估值增量 |

**🆕 催化剂新鲜度衰减**：

| 催化剂首次出现 | 权重系数 |
|:---|:---:|
| 今日首次出现 | ×1.0 |
| 昨日已出现（第2天） | ×0.85 |
| 3天前已出现 | ×0.70 |
| 5天前+ | ×0.50 |

同一条催化剂连续出现在多日推送中→权重衰减，避免重复定价。

### Step 6：风险标注（不禁筛）

所有品种都进排序，但标注风险：

| 条件 | 标注 | 含义 |
|:---|:---|:---|
| B≤2 | ⚠️ 供给刚性不足 | 地基本牢，赔率弱 |
| 催化剂=0 | ⚠️ 无扳机 | 不知道钱什么时候来 |
| C≤2 | ⚠️ 低确定性 | 假设为主，需验证 |
| 近5日>20% 或 近20日>40% | ⚠️ 短期涨幅过大 | 追高风险，等回调 |
| 当前价已超T1 | ⚠️ 预期已部分反映 | 安全边际收窄 |
| 当前价逼近T2 | ⚠️ 预期接近充分定价 | 上行空间有限 |
| B≥8 + G≥6 + C≥4 | 🏆 黄金赔率 | 三强共振 |

**不淘汰任何品种**——淘汰是老大自己判断的事，AI只排序+标注风险。

### Step 7：买入评级

| 综合分 | 评级 |
|:---:|:---|
| ≥8.0 | 🟢 **强力买入** — 三因子全强 |
| 6.5-7.9 | 🟢 **建议买入** — 强两因子 |
| 5.0-6.4 | 🟡 **可买入** — 赔率合理 |
| 3.5-4.9 | ⚪ **观察等待** |
| <3.5 | — **不推荐** |
