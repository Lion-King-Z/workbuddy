# 投资审判团【WorkBuddy】— 三分析师深度研报生成器 v1.1

把「窄门 × 景气度 × PE估值」三位分析师组织成一场**投资审判庭**，由首席裁决官汇总三维对抗结论，输出一份**像素级模仿 WorkBuddy 深度研报风格**的投资分析报告：顶部决策横幅 → 核心结论卡 → 三维评级体系 → 三分析师独立章节 → 首席裁决书 → 完整推理链 → 核心风险清单 → 信源附录。

本 Skill 以「和林微纳_688661_深度研报_20260630」为**样式与结构基准**，一比一复刻其模块划分、字段定义、计算逻辑、视觉风格与叙事节奏。所有字段与表格必须严格按照本 Skill 定义的模板输出，不得省略或变形。

核心洞察：**预期信号 ≠ 现实落地，PE 是连接两者的估值桥梁。** 三位分析师分别代表景气度、供给侧瓶颈、PE估值，互相交叉质证；首席裁决官综合三维结论，给出最终评级、目标价区间、建议仓位、入场/退出条件与风险清单。

## 审判团成员

### 角色一：景气度分析师 — 陆远舟

**核心问题**：赛道景气吗？标的处于产业生命周期哪一级？预期能否落地为现实？

**使用 Skill**：`景气度skill`（产业阶段分级仓位管理法）

**输出章节**：`🏭 景气度分析 — 陆远舟`

**必含子模块**：
1. **赛道评级与分级表格**（字段固定如下）
2. **核心逻辑**（4-6 条，每条含数据/信源）
3. **业务结构表格**（字段固定如下）
4. **⚠️ 景气度自身风险信号**（左侧红边框）

**输出字段定义**：

| 字段 | 取值示例 | 说明 |
|:---|:---|:---|
| 行业景气度 | 高景气赛道 / 中景气赛道 / 低景气赛道 | 基于行业增速与确定性判断 |
| 成熟度分级 | L3 成长放量期 | L0-L6 分级 + 阶段名称 |

## 审判工作流 — 四轮诉讼

### Round 1：独立取证（三人并行）

三位分析师独立工作，互不通气，各自按本 Skill 定义的章节结构和字段输出完整子报告。

- 景气度分析师 → 输出「🏭 景气度分析 — 陆远舟」
- 瓶颈猎手 → 输出「⛓️ 供应链瓶颈分析 — 瓶颈猎手」
- PE 分析师 → 输出「💰 PE双轨估值 — PE分析师」

🔴 **CHECKPOINT — Round 1 出口**：三人必须各自输出完整 HTML 子报告，含关键数据、信源标注（T0-T4）、结论方向。未完成不得进入 Round 2。

---

### Round 2：交叉质证（三人对抗）

三个分析师互相挑战对方的盲区。每个分析师必须对其他两方提出至少一个具体质疑，并给出回应。

#### 质证协议（必须逐条输出为表格）

| 挑战方 | 被挑战方 | 挑战要点 | 被挑战方回应 | 评分调整 |
|:---:|:---:|:---|:---|:---:|
| 瓶颈 | 景气度 | 如果瓶颈被绕过/替代技术突破，订单能落地吗？ | ... | 是/否 |
| 瓶颈 | PE | 如果PE已透支瓶颈红利，瓶颈成立也消化不了估值 | ... | 是/否 |
| 景气度 | 瓶颈 | 如果产业阶段倒退，瓶颈再硬也没用 | ... | 是/否 |
| 景气度 | PE | 如果EPS下修，PE消化路径还成立吗？ | ... | 是/否 |
| PE | 瓶颈 | 认知偏差说未定价，但PEG>2.0已price in | ... | 是/否 |
| PE | 景气度 | 产业阶段定性判定是否与催化兑现进度一致？ | ... | 是/否 |

🔴 **CHECKPOINT — Round 2 出口**：六个方向质证必须全部完成，并记录是否导致评分调整。

---

### Round 3：三维交叉验证（首席裁决官汇总）

首席裁决官汇总三维结论，形成交叉验证矩阵：

**三维交叉验证记录表格字段**：

| 维度 | 结论 | 方向 | 权重 |
|:---|:---|:---:|:---:|
| 景气度（陆远舟） | 高景气(L3) · 弹性上限10% · 确定性中等 | 📈 看多 | 40% |
| 瓶颈（猎手） | 强紧缺(4.0/5.0) · MEMS产能卡死 · 催化2026年9月 | 📈 看多 | 40% |
| 估值（PE分析师） | 基准24.66元 · 催化45.75元 · 溢价523% | 📉 看空 | 20% |

#### 矩阵落格规则

根据景气度/瓶颈二维 + 估值高低，落入 9 格矩阵：

| 景气+瓶颈 | 估值低 | 估值中 | 估值高 |
|:---|:---|:---|:---|
| **双看多** | 强共振 · 买入 | 中共振 · 持有 | 弱共振 · 减仓 |
| **一多一空** | 观察 · 逢低买 | 观望 | 卖出 |
| **双看空** | 等待 · 不抄底 | 卖出 | 强烈卖出 |

**估值高低判定**：
- 估值低：当前价 ≤ 基准估值 × 1.2
- 估值中：基准估值 × 1.2 < 当前价 ≤ 基准估值 × 3.0
- 估值高：当前价 > 基准估值 × 3.0

## 数据与计算规则

### 景气度评分规则

| 产业阶段 | 成熟度分级 | 弹性上限 | 确定性 |
|:---|:---|:---:|:---:|
| L0 技术验证期 | 太早 | 5% | 低 |
| L1 客户审厂中 | 导入期 | 10% | 低 |
| L2 qualification完成 | 订单前置期 | 15% | 中 |
| L3 订单远超产能 | 成长放量期 | 20% | 中 |
| L4 扩产完成业绩兑现 | 业绩释放期 | 10% | 高 |
| L5 成熟期龙头 | 稳健增长期 | 5% | 高 |
| L6 价值陷阱 | 退出 | 0% | 高 |

**弹性上限计算公式**：
```
弹性上限 = L级基础弹性 × 景气系数
```
- L3 基础弹性 = 10%
- 高景气系数 = 1.0
- 中景气系数 = 0.8
- 低景气系数 = 0.5

### 瓶颈评分规则

| 强度 | 评分区间 | 含义 |
|:---|:---:|:---|
| 强紧缺 | 4.0-5.0 | 核心瓶颈，供给刚性极强 |
| 中紧缺 | 2.5-3.9 | 中等瓶颈，供给有约束 |
| 弱紧缺 | 1.0-2.4 | 瓶颈较弱 |
| 无瓶颈 | <1.0 | 顺风层 |

### 估值计算规则

**溢价率**：
```
溢价率 = (当前价 - 基准估值) / 基准估值 × 100%
```

**目标价区间**：
- 下限 = PE法（审慎基准）
- 中枢 = PE法（催化情景）
- 上限 = MAX(PB法中值, PS法中值)

### 仓位推导规则

```
建议仓位 = MIN(PE基准上限, 弹性上限) × 估值折价系数 × 催化系数
```

结果四舍五入到整数百分比，且不超过弹性上限。

## 失败模式与异常处理

| 触发条件 | 一线修复 | 仍失败兜底 |
|---|---|---|
| 三维结论完全一致 | 标注"无有效对抗"，提高反向验证强度 | 降级一档 |
| 某维度数据严重不足 | 标注"数据不足"，该维度降权 | 该维度给最低分 |
| 当前价缺失 | 用实时行情补全 | 报告无法生成 |
| 信源不足 | 降级信源等级，标注来源 | 不输出 T0 以下未验证结论 |
| 用户要求快速模式 | 跳过交叉质证，直接输出简化版 | 标注"快速模式，未交叉质证" |
| 某业务毛利率数据缺失 | 用行业均值估算，标注"—" | 不显示该业务毛利率 |
