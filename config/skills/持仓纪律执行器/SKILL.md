# 持仓纪律执行器 Skill v1.0

**定位**：持仓纪律的**执行检查器**。不是复盘，不是分析——是纪律执行。

**核心原则**：
1. 纪律不是"复盘后再决定"——到了价位就执行
2. 止盈看估值到位+催化兑现，止损看基本面证伪+风险恶化
3. 加仓看催化剂超预期+回调到安全区，减仓看涨幅过大+估值偏高
4. 只提建议和风险提示，不替老大做决定

---

## 一、触发条件

- 用户问"该不该卖""该不该加仓""止盈止损""减仓条件"
- 每日个人复盘调用（作为纪律检查输入）
- 持仓标的触发关键价位时主动提醒

---

## 二、所需输入

| 数据项 | 来源 | 用途 |
|:---|:---|:---|
| 持仓清单（标的+成本价+仓位比例） | 用户提供/每日个人复盘 | 纪律执行的基础 |
| T1/T2/T3目标价 | pe-valuation-analyst | 止盈/止损价位参考 |
| 当前价/近5日/20日涨幅 | tdx-connector | 实时价位监控 |
| 综合风险评级 | 个股风险扫描器 | 风险恶化判断 |
| B评分+反向验证总分 | 产业链瓶颈投研skill | 基本面证伪判断 |
| 催化剂兑现率 | pe-valuation-analyst Step 3 | 催化兑现判断 |
| 财务健康标签 | pe-valuation-analyst v2.5 Step 4.5 | 财务恶化判断 |

## 三、四大纪律触发条件

### 3.1 止盈触发条件

| 条件 | 触发动作 | 优先级 |
|:---|:---|:---:|
| 当前价 ≥ T2目标价 | 建议止盈50%仓位 | 🔴高 |
| 当前价 ≥ T3目标价 | 建议止盈全部仓位 | 🔴高 |
| 催化剂兑现率 > 80% | 建议止盈30-50%仓位 | 🟡中 |
| 近60日涨幅 > 50% 且 估值已超T1 | 建议分批止盈 | 🟡中 |
| 近60日涨幅 > 80% | 建议止盈至少50% | 🔴高 |
| PE(TTM) > 行业90%分位 + PEG > 2.0 | 建议减仓50% | 🟡中 |

### 3.2 止损触发条件

| 条件 | 触发动作 | 优先级 |
|:---|:---|:---:|
| 当前价 ≤ T1目标价 × 0.9（跌破T1的90%） | 建议止损，标注"估值底已破" | 🔴高 |
| 瓶颈反向验证总分 < 2.0 | 建议止损，标注"瓶颈证伪" | 🔴高 |
| 个股风险扫描器评级升至🔴高风险 | 建议止损或大幅减仓 | 🔴高 |
| B评分从≥6降至≤3 | 建议止损，标注"基本面恶化" | 🔴高 |
| 财务健康标签新增≥2个🔴 | 建议止损，标注"财务恶化" | 🟡中 |
| 近20日跌幅 > 25% 且 无催化改善 | 建议止损，标注"趋势破位" | 🟡中 |
| 产业阶段从3级降至1级 | 建议止损，标注"阶段倒退" | 🔴高 |

### 3.3 加仓触发条件

| 条件 | 触发动作 | 优先级 |
|:---|:---|:---:|
| 催化剂超预期兑现 | 建议加仓20-30% | 🟡中 |
| 当前价回调至T1以下 | 建议加仓，标注"安全边际恢复" | 🟡中 |
| 个股风险扫描器评级从🟠降至🟢 | 建议加仓，标注"风险解除" | 🟢低 |
| B评分从≤3升至≥6 | 建议加仓，标注"瓶颈确认" | 🟡中 |
| 产业阶段从1级升至3级 | 建议加仓，标注"黄金区间启动" | 🔴高 |
| 近20日回调>15% 但 基本面未恶化 | 建议分批加仓，标注"情绪错杀" | 🟡中 |

### 3.4 减仓触发条件

| 条件 | 触发动作 | 优先级 |
|:---|:---|:---:|
| 近5日涨幅 > 20% | 建议减仓20-30%，标注"短期过热" | 🟡中 |
| 近20日涨幅 > 40% | 建议减仓30-50%，标注"中期过热" | 🟡中 |
| 当前价逼近T2（距T2<10%） | 建议减仓，标注"接近充分定价" | 🟡中 |
| 催化剂已兑现但股价继续大涨 | 建议减仓，标注"透支远期" | 🟢低 |
| 融资盘进入警戒区 | 建议减仓，标注"融资盘抛压" | 🟡中 |
| 大股东质押比例升至>50% | 建议减仓，标注"控制权风险" | 🟡中 |

## 六、与每日个人复盘的衔接

| 每日个人复盘 | 持仓纪律执行器 |
|:---|:---|
| 做操作评价（昨天买了/卖了对不对） | 做纪律执行判断（今天该不该卖/买） |
| 牵牛五维指标 D1-D5 | 四大纪律触发条件 |
| 持仓风险仪表盘 | 纪律执行检查表 |
| 复盘导向 | 执行导向 |

每日个人复盘可调用本Skill获取纪律建议，然后由复盘Skill做综合评价。

---

## 七、注意事项

- 本 Skill 不提供"买入建议"——买入找窄门筛股
- 本 Skill 不做"深度分析"——深度找投资审判团
- 本 Skill 只做纪律执行检查：到了价位没有？基本面变没变？风险恶化没有？
- 严格遵循决策边界铁律：只提建议，不替老大执行
- 数据缺失时标注"数据缺失，无法判断"，不强行给出纪律建议
