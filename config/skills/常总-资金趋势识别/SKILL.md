# 常总-资金趋势识别

## R — 原文

> 价格是结果，资金才是因。在这个市场里，99%的人在看价格，真正的赢家只看一件事——资金流向 + 量能结构。市场没有量，就没有行情。没量 → 不看；放量 → 关注；持续量 → 确认主线；量衰竭 → 立即撤退。
>
> — 常士杉，《资金流向与量能结构——主线抓取核心体系》

---

## I — 方法论骨架

本工作流把“钱往哪里走、有没有持续性、能不能进场、要不要撤退”四个问题串成一条可执行的判断链。它先用量能区分真假异动，再用7-13个交易日的量能维持率验证资金是否锁仓，随后用周线/日线共振判断趋势级别，最后落到五日战法的具体买卖点，并叠加重大事件提前识别的盘面信号。整个流程只做一件事：让交易跟随资金，而不是跟随价格或新闻。

---

## A1 — 书中的应用

1. **主线确立案例**：某板块连续三日放量，随后7-13个交易日成交量始终维持在爆发日峰值的55%以上，周线同时放量突破。结论：资金锁仓，主线成立，进入五日战法执行阶段。
2. **诱多识别案例**：个股单日巨量涨停，但随后三日量能迅速萎缩至峰值55%以下，价格横盘滞涨。结论：主力未锁仓，一波流概率高，不进场、不追高。
3. **重大事件提前捕捉案例**：大盘下跌期间，某板块多只个股同时出现“无新闻但成交量持续放大、盘口拒绝下跌、板块同步异动”三重信号。结论：资金提前写入盘面，应立刻研究背后是否存在未公开催化，并准备五日战法二次进场条件。

---

## A2 — 触发场景

1. 用户问“这个板块/个股有没有放量”“资金是不是在进场”。
2. 用户问“能不能追突破”“突破是不是诱多”。
3. 用户问“资金在往哪个方向走”“主线有没有成立”。
4. 用户问“有没有重大事件提前异动”“盘面有没有风声”。
5. 用户已经持有某标的，问“要不要继续持有”“什么时候该走”。

语言信号：
- “放量了吗”“有没有量”“量能怎么看”
- “能不能追”“是不是真突破”“会不会诱多”
- “资金在往哪里走”“主线确认了吗”
- “提前异动”“盘面异常”“没消息但放量”
- “跌破多少走”“跌破5日线怎么办”

与相邻 skill 区分：
- 与 `常总-赛道稀缺卡位评估` 的区别：赛道稀缺卡位评估回答“这个行业/公司值不值得长期陪伴”，本 skill 回答“资金现在是否在持续流入、短期趋势是否成立”。
- 与 `常总-估值与逆向买入` 的区别：估值与逆向买入回答“什么价格算便宜、该不该左侧抄底”，本 skill 回答“资金是否已经用行动确认方向、右侧如何跟进”。
- 与 `常总-持有纪律与公司排雷` 的区别：持有纪律回答“持有中遇到公司风险或市场波动如何处置”，本 skill 回答“进场前/进场后如何根据资金趋势判断去留”。

## E — 可执行步骤

当 skill 被激活后，按以下步骤执行。每一步必须输出“判断结论”和“下一步动作”，不得跳过判停条件。

### 步骤1：量能基础判断

判断当前价格运动是由资金推动还是散户波动。

- 输入：标的近5-10个交易日的成交量、成交额、价格K线。
- 判断标准：
  - **放量上涨**：成交量显著高于近期均值，价格同步上涨 → 主力建仓/抢筹信号。
  - **放量下跌**：成交量显著放大，价格下跌 → 主力换手/洗盘，需结合位置和后续量能判断。
  - **无量波动**：成交量萎缩，价格小阴小阳或横盘 → 散户自娱自乐，无参与价值。
- 完成标准：明确当前量价组合属于“放量上涨 / 放量下跌 / 无量波动”三类之一，并说明资金性质。
- 判停条件：
  - 若属于**无量波动**，直接输出“不看不买，等待放量”，本 skill 流程终止。
  - 若属于**放量上涨**，进入步骤2。
  - 若属于**放量下跌**，进入步骤3的趋势状态判断（区分底部巨量还是无量下跌）。

### 步骤2：资金持续性判断（主线确认）

判断放量是否只是单日诱多，还是资金真正持续流入。

- 输入：放量爆发日成交量V_peak；爆发日后第1-13个交易日每日成交量V_t。
- 核心规则：
  - 巨量爆发后，未来7-13个交易日内，成交量必须 **≥ 峰值V_peak的55%**。
  - 若连续3日量能 < 55% → 主力撤退，高概率暴跌/崩盘，直接离场或放弃。
- 完成标准：
  - 若7-13日内量能维持率 ≥ 55% → 输出“资金锁仓，主线成立”，进入步骤4。
  - 若不满足 → 输出“一波流/诱多，不进场”，本 skill 流程终止。
- 判停条件：资金不持续则立即判停，禁止因单日放量冲动进场。

### 步骤3：趋势状态判断（周线级别与底部结构）

在日线节奏之外，确认更大级别的趋势是否配合。

- 输入：标的周线成交量、周线价格结构；若为下跌后的放量，需要日线位置。
- 判断标准：
  - **周线放量突破**：周线成交量放大并突破关键位置 → 中期行情启动，资金确认。
  - **周线量能衰竭**：周线成交量持续萎缩或价格滞涨 → 主升浪结束，趋势危亡。
  - **底部巨量**：长期下跌后突然放出历史级巨量，价格止跌或震荡 → 主力建仓完成，可进入步骤4。
  - **无量下跌**：连续13个交易日量能持续萎缩，价格阴跌 → 没人接盘、主力放弃，进入“死亡区间”，直接放弃。
- 完成标准：明确当前处于“周线量在/周线量失/底部巨量/无量下跌”四类之一，并给出趋势级别判断。
- 判停条件：
  - 若**周线量失**或**无量下跌** → 输出“趋势不在，不进场/应离场”，本 skill 流程终止。
  - 若**周线量在**或**底部巨量** → 进入步骤4。

### 步骤4：五日战法执行（进场、持有、离场、二次进场）

当资金持续、趋势配合后，用五日战法给出具体交易规则。

#### 4.1 进场条件（三者必须同时满足）

- 输入：日线收盘价、5日均线、连续三日成交量、七日内最大单日成交量。
- 条件：
  1. **站上5日线**：收盘价在5日均线上方（非盘中）。
  2. **连续三天放量**：三个交易日成交量逐日放大或维持活跃水平，一天都不能少。
  3. **七天内出现巨量**：单日成交量 ≥ 前一个交易日均量的 **1.45倍**。
- 完成标准：三个条件同时满足，输出“主力进场确认，可以建仓”；采用3-2-2-2阵型，首笔约35%仓位试探。
- 判停条件：任一条件不满足 → 输出“不上5日线不看、不放量不买、没巨量不跟”，等待信号，不进场。

#### 4.2 持有纪律

- 输入：每日收盘价、5日均线数值。
- 规则：收盘价永远不低于5日线的 **7.5%**。
- 举例：5日线 = 10元，7.5% = 0.75元，收盘价 ≥ 9.25元 → 安心持有，不理盘中急跌。
- 完成标准：持仓期间每日收盘后复核，若未跌破7.5% → 继续持有。
- 判停条件：收盘价跌破5日线7.5% → 进入4.3卖出纪律第一步。

#### 4.3 卖出纪律

- 第一步：收盘价跌破5日线7.5%的**第一时间** → 减仓一半，不犹豫、不等反弹。
- 第二步：继续下跌，触及**20%止损线** → 清仓离场，一股不留。
- 完成标准：触发任一条件即执行相应卖出动作，输出减仓/清仓信号。
- 判停条件：清仓完成后，本 skill 交易流程终止；后续若出现二次进场信号，重新进入4.4。

#### 4.4 纠错与二次进场

- 前提：止损后，股价放量反抽，重新站回5日线附近。
- 条件（必须同时满足）：
  1. 反抽时放出巨量（≥ 1.45倍）。
  2. 收盘价距离5日线的幅度 ≤ **4.5%**。
- 举例：5日线 = 10元，4.5% = 0.45元，收盘价 ≥ 9.55元 → 果断重新进场。
- 完成标准：同时满足则输出“二次进场信号确认，可重新建仓”；不满足则输出“放弃，不再追”。
- 判停条件：错过二次进场后不再追高，本 skill 流程终止。

### 步骤5：重大事件提前识别

在步骤1-4执行过程中，始终并行观察重大事件前盘面痕迹。

- 输入：标的成交量异动、价格相对大盘/板块走势、盘口逐笔数据、板块内联动、期权数据（如有）、消息面热度。
- 七大蛛丝马迹：
  1. **异常提前放量**：股价未涨，但成交量突然持续放大，无新闻、无公告，量能明显高于历史均值。
  2. **逆势抗跌或逆势上涨**：大盘跌、该股稳；同行业普跌、个股独自走强；尾盘频繁被资金托起。
  3. **盘口隐形大单吸筹**：分笔小单持续买入，买一/买二不断被秒吃掉，算法化隐蔽建仓。
  4. **股价拒绝下跌**：利空不跌、调整缩量、下跌立刻被拉回，筹码被高度控盘。
  5. **板块同步异动**：同一产业链/主题多只个股同时放量，群体共振。
  6. **期权与衍生品异常**：看涨期权暴增、隐含波动率飙升、深度虚值期权被大量买入。
  7. **消息面异常沉默**：媒体安静、官方无解释、市场充满不确定性，真正的信息掌握在少数人手中。
- 完成标准：识别出至少一项蛛丝马迹，并给出“是否存在重大事件提前异动”的初步判断。
- 判停条件：
  - 若**异常放量 + 板块联动 + 抗跌结构**三者同时出现 → 输出“重大事件前高概率信号，应结合五日战法准备进场/二次进场”，回到步骤4做交易计划。
  - 若仅有单一信号 → 输出“需继续观察，不单独作为进场依据”，不影响步骤1-4的正常执行。
  - 若无任何信号 → 输出“当前无重大事件提前异动迹象”，继续执行常规流程。

## B — 边界

### 不要在以下情况使用此 skill
- 判断一个行业/公司是否具备长期赛道价值（使用 `常总-赛道稀缺卡位评估`）。
- 评估当前股价是否便宜、是否适合左侧逆向买入（使用 `常总-估值与逆向买入`）。
- 持有期间出现公司基本面风险、财务暴雷或黑天鹅事件需要排雷（使用 `常总-持有纪律与公司排雷`）。
- 纯短线技术博弈、高频交易或完全没有成交量数据的标的。

### 作者在书中警告的失败模式
- **看到单日放量就冲**：巨量爆发后若7-13日量能不能维持峰值的55%，多是诱多，进场容易被套。
- **无量下跌中抄底**：“底部无量 = 继续跌”，没有历史级巨量配合的底部不可信。
- **跌破7.5%还等反弹**：作者明确“不要犹豫，不要等反弹”，纪律不执行会从小亏变成大亏。
- **忽略周线只看日线**：日线看节奏，周线看命运；周线量失意味着趋势危亡，必须离场。
- **追新闻而不是追资金**：重大事件出来后再追，往往是接盘；真正的大机会在新闻之前的资金异动里。

### 作者的盲点 / 时代局限
- 文章以A股、游资和机构行为为主要语境，对美股、港股、加密货币等市场流动性结构差异覆盖不足。
- 五日战法高度依赖5日均线和量价规则，对极端流动性危机、连续跌停、长期停牌等异常状态缺乏应对。
- 期权与衍生品异常信号对普通散户可获取性有限，实际操作中可能需要借助专业数据终端。
- 文中“主力”“庄家”等表述带有A股特定市场文化色彩，在机构化、量化主导的市场中资金行为可能更复杂。
- 1.45倍、55%、7.5%、20%、4.5%等阈值是经验参数，未经过严格的统计回测，使用者应根据自身风格和市场环境调整。

### 容易混淆的邻近方法论
- **五日战法 vs 一般均线突破**：五日战法不是简单的站上5日线，而是“站上5日线 + 连续三天放量 + 七天内巨量”三条件同时满足。
- **资金趋势 vs 技术趋势**：技术趋势看价格形态，资金趋势看成交量和资金持续；价格可以骗人，成交量更难骗人。
- **重大事件提前识别 vs 消息面驱动交易**：前者是“资金先于消息”，后者是“消息出来后追涨”；前者是预判，后者是跟风。
- **放量上涨 vs 放量下跌**：两者都放量，但资金性质不同；放量上涨可能是建仓，放量下跌可能是洗盘或出货，需要结合位置和后续量能判断。
