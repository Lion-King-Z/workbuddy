# 浑水电话会 — 分析框架参考

## 电话会类型与关注重点

| 电话会类型 | 核心关注点 | 可疑信号 |
|-----------|-----------|----------|
| **行业策略会** | 产业链上下游交叉验证、景气度共识/分歧 | 过于一致的乐观预期 |
| **个股调研** | 产能、订单、毛利率、客户结构 | 回避回答核心问题 |
| **宏观专题** | 政策预期、流动性、配置方向 | 过度依赖单一假设 |
| **紧急解读** | 突发事件影响链、供需再平衡时间 | 短期情绪放大 |

## 跨期信号跟踪（3日→7日→整月）

当同一环节在多日电话会中被反复提及时，建立时间序列：

```
Day 1: 提及 — 首次暴露
Day 3: 第二次提及 — 趋势确认
Day 7: 多次提及 — 强信号，可整理为独立研究线索
Day 30: 月度汇总 — 判断景气度持续性
```

## 高频词汇与景气度映射

| 电话会高频词 | 景气度信号 | 操作含义 |
|-------------|-----------|----------|
| "订单排到" / "满产" / "产能不够" | ↑↑ 强上行 | 关注供给瓶颈环节 |
| "库存消化" / "去库存" / "需求疲软" | ↓↓ 下行 | 回避周期高位标的 |
| "客户在等" / "观望" / "招标推迟" | → 持平偏弱 | 等待拐点信号 |
| "涨价" / "提价成功" / "价格传导" | 🔥 定价权增强 | 核心投资信号 |
| "扩产" / "新产能" / "爬坡" | ⚠ 供给释放 | 关注竞争格局变化 |
| "进口替代" / "国产化" / "自主可控" | 🏭 结构机会 | 关注国产份额提升 |

## 买方提问质量评估

高价值提问特征：
- 追问具体数据（不是"行业前景怎么样"）
- 要求拆分量价（"量增还是价增"）
- 质疑乐观假设（"如果下游砍单呢"）
- 跨行业对比（"和XX行业比呢"）

低价值提问：直接忽略，不做分析依据。
