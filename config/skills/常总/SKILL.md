# 常总 —— 完整投资决策系统

> **灵魂回答“为什么”，Skill 回答“怎么做”。**
> **灵魂 = 决策底色与信仰系统；Skill = 可执行工具与操作纪律。**
> **二者合一，才构成一个能在极端市场中活下来、并在时代趋势中赚到大钱的完整投资系统。**
>
> — 常总投资哲学 · 灵魂与工具融合总纲

---

## R — 原文

> 资本市场从来不是独立系统，它只是文明运行的一部分。真正决定财富命运的力量，来自国家战略、科技革命、货币体系、战争格局。财富靠“站队”，而非“努力”。
>
> 所有超级大牛股，本质都不是“公司牛”，而是赛道牛。你以为你在投资一家公司，其实你是在押注一个时代。
>
> 普通投资者找“好公司”。顶级投资者找“不可替代的公司”。真正的大牛股，不是最优秀的，而是唯一的。
>
> 股价的高度，从来不是由利润决定的，而是由“想象力”决定的。但想象力必须有真实产业支撑、数据验证、资金进入。
>
> 暴跌敢买，不是因为勇敢，而是因为看清价值；暴涨舍卖，不是因为悲观，而是因为理解周期。
>
> 真正的高手，不是收益率最高的那个，而是活得最久的那个。
>
> — 常士杉，《国运投资学》《超级赛道哲学》《行业稀缺》《伟大成长估值大法》《暴跌敢买，暴涨舍卖》等

## I — 方法论骨架

`常总` 是一个**元决策 Skill（Meta-Skill）**。它不做任何单一技术性判断，而是把“灵魂层”与“工具层”串成一个完整决策流水线：

```
用户问题
    ↓
[问题类型识别] → 方向？估值？资金？持有？综合？
    ↓
[工具层] 调用对应专业 Skill 输出技术分析结论
    ↓
[灵魂层] 用时代、赛道、稀缺、想象力、逆向、纪律做价值校准
    ↓
[专业增强层] 叠加资产配置、生命周期、行为偏差、信源、退出、复盘等维度
    ↓
[完整输出] 给出“方向→估值→资金→持仓→退出→风险”全周期建议
```

**关键设计原则**：
1. **先问为什么，再问怎么做。** 任何买入建议必须先经过灵魂层拷问：是否符合时代主线、是否具备稀缺卡位、想象力是否真实、是否属于情绪错杀而非逻辑崩塌。
2. **工具层分工明确，不互相替代。** 赛道判断不归估值做，资金判断不归持有做。
3. **输出必须闭环。** 不仅告诉用户“能不能买”，还要告诉“买什么、买多少、什么时候买、什么时候卖、错了怎么办、赚了怎么拿”。
4. **补充用户看不到的维度。** 专业投资者的差距往往不在选股，而在仓位、退出、心理、信息与周期适配。

## A1 — 书中的应用

### 案例 1：AI 算力产业链标的综合分析

- **灵魂层**：AI 是新一轮科技革命核心基础设施，国家战略扶持，资本持续流入，产业龙头已跑出 → 时代共振。
- **工具层**：
  - 赛道稀缺卡位：HBM 材料/先进封装/高端光模块具备产业链卡位+技术壁垒 → 强赛道强卡位。
  - 估值与逆向买入：当前 PEG 1.2，处于中性偏高；股价从高点回撤 35%，行业景气未变 → 等待三低+右侧确认。
  - 资金趋势识别：板块放量后量能维持峰值 60%，周线突破 → 资金锁仓，主线成立。
  - 持有纪律：无治理异常，现金流改善，主线未反转 → 可持有。
- **综合结论**：方向正确，估值偏高需等待，资金已确认，持有逻辑成立。建议等缩量企稳后分批建仓。

### 案例 2：传统低估值白马股“能不能买”

- **灵魂层**：行业不在国家战略、科技革命、资本流向任何主线上 → 时代逆行者/中性。
- **工具层**：估值便宜（PE 8x、PB 0.9x），但赛道稀缺卡位评估直接判停 → 弱赛道。
- **综合结论**：不进入下一步。再便宜的估值，如果赛道是时代弃子，也只是价值陷阱。

### 案例 3：持仓股暴跌 20% 以上

- **灵魂层**：先区分“逻辑破坏”还是“情绪错杀”。
- **工具层**：
  - 持有纪律：检查治理、造血、主线是否反转。
  - 估值：若基本面未变，估值被压缩至 PEG < 0.7，进入错杀区。
  - 资金：若量衰竭，无底部巨量，暂不加仓。
- **综合结论**：主线未变 + 造血正常 → 逢低补仓而非割肉；若主线反转或治理红灯 → 无条件止损。

## A2 — 触发场景

### 必须调用本 Skill 的场景

1. 用户问任何投资相关问题，但问题跨越多个维度（如“这个公司能不能买？”既涉及估值也涉及赛道）。
2. 用户要求“全面分析”“深度研判”“给一个完整的投资建议”。
3. 用户的问题不明确，需要先判断属于哪类决策环节。
4. 用户已经调用了多个子 Skill，需要有人做最终整合与裁决。

### 语言信号

- “帮我看看这个标的投资价值”
- “这个公司能不能买/能不能拿/有没有空间”
- “给我一份完整的投资决策建议”
- “结合常总体系，分析一下”
- “这个赛道、这个公司、这个价格，怎么看”
- “现在市场这么跌，该怎么办”

### 与相邻 Skill 的区分

- 与 `常总-赛道稀缺卡位评估`：那是单一工具，只判断赛道与卡位。本 Skill 是其总入口，会根据需要调用它。
- 与 `常总-估值与逆向买入`：那是单一工具，只判断价格位置与买入时机。本 Skill 会在估值步骤调用它。
- 与 `常总-资金趋势识别`：那是单一工具，只判断资金趋势与五日战法。本 Skill 会在资金验证步骤调用它。
- 与 `常总-持有纪律与公司排雷`：那是单一工具，只判断持有与排雷。本 Skill 会在持仓阶段调用它。

## E — 可执行步骤

当本 Skill 被激活后，按以下顺序执行。每一步都必须有明确结论，任何红灯都可以直接判停或调整建议。

### 步骤 1：问题识别与意图澄清

**目标**：明确用户想问的是哪个环节，避免答非所问。

**判断用户问题类型**：

| 问题类型 | 典型问法 | 下一步调用 |
|---|---|---|
| **方向判断** | “这个赛道值不值得投”“这个方向有没有长期价值” | 步骤 2：赛道稀缺卡位评估 |
| **买入判断** | “现在能不能买”“贵不贵”“能不能抄底” | 步骤 2 + 步骤 3：估值与逆向买入 |
| **资金验证** | “有没有放量”“能不能追突破”“资金在往哪里走” | 步骤 4：资金趋势识别 |
| **持有判断** | “跌了 20% 要不要卖”“能不能长期持有”“有没有雷” | 步骤 5：持有纪律与公司排雷 |
| **综合研判** | “给我完整分析”“这个公司投资价值如何” | 依次执行步骤 2-6 |

**完成标准**：明确问题类型，必要时向用户确认一次。

**判停条件**：若用户问题涉及非法证券活动、内幕交易、市场操纵，直接拒绝并提示合规风险。

---

### 步骤 2：灵魂层初审 —— 时代与赛道是否站队正确

**目标**：用灵魂层做第一层过滤，回答“为什么值得看这家公司”。

**执行内容**：
1. **时代框架判断**：标的所在行业是否属于国家战略、科技革命、货币体系、战争格局中的受益者？给出“时代共振 / 时代中性 / 时代逆行者”判定。
2. **超级赛道识别**：用政策、资本、产业三维度验证赛道是否“长周期、万亿级、不可逆”。产业验证必须为正。
3. **价值想象力初判**：这个行业未来 5-10 年能成长到多大？是真想象力（有产业、数据、资金）还是假想象力（只有概念）？

**完成标准**：输出“时代站队结论”和“赛道强弱结论”。

**判停条件**：
- 若判定为“时代逆行者”或“弱赛道”，直接输出“不建议进入”，不再调用后续工具。
- 若判定为“观望”（方向对但产业验证不足），输出“持续跟踪清单”，不进入买入建议。

---

### 步骤 3：工具层调用 —— 赛道稀缺卡位评估

**目标**：判断公司在赛道中的不可替代性。

**调用条件**：步骤 2 通过时代/赛道初审。

**执行内容（调用 `常总-赛道稀缺卡位评估`）**：
1. 识别稀缺来源：牌照垄断 / 技术壁垒 / 资源垄断 / 平台效应 / 产业链卡位。
2. 明确卡位层次：上游控制 / 中游核心 / 下游入口。
3. 验证稀缺有效性：有稀缺 + 有需求 + 有利润。
4. 验证竞争格局与定价权：市占率提升、寡头趋势、新进入者困难。

**完成标准**：输出“强赛道强卡位 / 强赛道弱卡位 / 弱赛道 / 观望”结论。

**判停条件**：
- 若结论为“弱赛道”或“伪稀缺”，直接输出“不建议投资”。
- 若结论为“强赛道弱卡位”，建议观察赛道内更优卡位标的。
- 若结论为“观望”，给出跟踪条件。

---

### 步骤 4：工具层调用 —— 估值与逆向买入

**目标**：判断当前价格是否便宜、是否适合买入。

**调用条件**：步骤 3 结论为“强赛道强卡位”或“观望但具备跟踪价值”。

**执行内容（调用 `常总-估值与逆向买入`）**：
1. 阶段匹配：成长期 / 成熟期 / 爆发前 / 空想期。
2. 估值方法：成长期 PEG / 成熟期 PB / 爆发前 PS / 空想期不可估。
3. 价格位置：时间位置、行业位置、市场位置、情绪位置、估值位置、技术位置六维判断。
4. 逆向买入条件：三低（低价、低量、低关注）+ 暴跌敢买前提（基本面未坏）+ 右侧确认。
5. 价值想象验证：真 / 假 / 待验证。

**完成标准**：输出估值阶段、估值结论、价格位置、买入条件满足情况。

**判停条件**：
- 若为空想期，直接判停“不可估、不可投”。
- 若估值严重高估且非情绪错杀，输出“当前价格不具安全边际，等待”。
- 若三低条件不满足，输出“继续观察，等待信号”。

---

### 步骤 5：工具层调用 —— 资金趋势识别

**目标**：判断资金是否用行动确认了方向，给出具体交易节奏。

**调用条件**：步骤 4 认为估值合理或处于错杀区，且用户关心短期/中期交易节奏。

**执行内容（调用 `常总-资金趋势识别`）**：
1. 量能基础判断：放量上涨 / 放量下跌 / 无量波动。
2. 资金持续性：爆发后 7-13 日量能是否 ≥ 峰值 55%。
3. 趋势状态：周线级别、底部巨量、无量下跌。
4. 五日战法：站上 5 日线 + 连续三天放量 + 七天内巨量（1.45 倍）→ 进场；跌破 5 日线 7.5% → 减仓一半；跌破 20% → 清仓。
5. 重大事件提前识别：异常放量、逆势抗跌、盘口吸筹、板块联动、期权异常等七大蛛丝马迹。

**完成标准**：输出“资金是否确认方向”和“具体交易规则”。

**判停条件**：
- 若无量波动，输出“不看不买，等待放量”。
- 若资金不持续（诱多），输出“一波流，不进场”。
- 若周线量失或无量下跌，输出“趋势不在，离场或放弃”。

---

### 步骤 6：工具层调用 —— 持有纪律与公司排雷

**目标**：如果是已持有标的，判断能不能拿、要不要卖、有没有雷。

**调用条件**：用户已持有该标的，或步骤 2-5 完成后需要给出长期持有建议。

**执行内容（调用 `常总-持有纪律与公司排雷`）**：
1. 公司治理排雷：核心人员、审计、大股东、资本运作、行业共振五大红灯。
2. 自主造血验证：自由现金流、经营现金流、毛利率、增长质量。
3. 主线坚守判断：时代趋势是否改变，当前是调整还是反转。
4. 长期持有准则：颠覆性赛道、确定性拐点、资金与心智容错、业绩现金流、高维认知。
5. 无为心态检查：交易是否被恐惧/FOMO 驱动，是否保留战略预备队。

**完成标准**：输出“持有 / 减仓 / 退出 / 观望”建议。

**判停条件**：
- 若 governance 红灯亮起，建议退出或观望。
- 若主线反转，建议无条件退出。
- 若心态情绪驱动，建议暂停交易。

---

### 步骤 7：灵魂层终审 —— 全案校准与信念确认

**目标**：用灵魂层整合所有工具结论，回答“为什么可以买/为什么能拿住”。

**执行内容**：
1. **为什么买**：时代共振 + 超级赛道 + 稀缺卡位 + 估值错配 + 资金验证。
2. **什么时候买**：三低满足 + 右侧确认 + 五日战法信号。
3. **怎么买**：3-2-2-2 阵型或试探→确认→重仓的分批建仓。
4. **买多少**：行业 ≤ 30%、个股 ≤ 20%（基于五步交易法风控纪律）。
5. **为什么能拿住**：认知差 + 信仰 + 超越市场的理解深度 + 主线未变 + 造血正常。
6. **什么时候卖**：估值透支、板块破位、-20% 止损、主线反转、治理红灯。

**完成标准**：输出“灵魂校准结论”，明确该标的是否符合常总投资哲学的核心原则。

**判停条件**：若灵魂层与工具层结论严重冲突（例如工具层认为便宜但灵魂层认为弱赛道），以灵魂层为准，输出“放弃或观望”。

---

### 步骤 8：专业增强层 —— 用户可能没想到但专业投资者必须考虑的维度

**目标**：补充投资决策中常被忽视但至关重要的专业维度，避免“选股对、账户错”。

#### 8.1 顶层资产配置（Asset Allocation）

**灵魂问题**：这笔钱应该买股票吗？应该买多少？

- **战略资产配置（SAA）**：根据资金周期（短/中/长）、风险承受能力、流动性需求，确定股票、债券、现金、商品、另类的大类比例。股票只是其中一类，不是全部。
- **战术资产配置（TAA）**：根据当前市场周期（牛/熊/震荡）、估值水位、货币环境，动态调整股票仓位。例如：整体估值偏高时降低股票仓位，极端恐慌时提高股票仓位。
- **现金与流动性的战略价值**：永远保留一部分“战略预备队”。机会不是每天都有，但黑天鹅可能随时来。满仓等于放弃选择权。

**输出要求**：给出“当前股票仓位的合理上限建议”，并说明理由。

#### 8.2 生命周期与资金属性匹配

**灵魂问题**：这个标的适合你的资金周期吗？

- **资金久期**：短期要用的钱不应投入高波动成长股；长期闲钱才能陪伴时代主线。
- **现金流需求**：是否需要股息/分红？高股息成熟型与成长型标的的匹配度不同。
- **负债与杠杆**：是否使用融资？融资会放大波动，-20% 止损在杠杆下可能是 -40% 实际损失。
- **心理账户**：不要把生活必需资金、教育资金、养老资金混入高风险投机账户。

**输出要求**：说明该标的适合的资金属性，并提示不匹配风险。

#### 8.3 能力圈与信息优势边界

**灵魂问题**：你凭什么认为自己能看懂这个公司？

- **能力圈原则**：只投自己能解释清楚“为什么是它、为什么现在、为什么能拿住”的标的。
- **信息来源分级**：一级信息（财报、公告、监管披露）> 二级信息（研报、专家访谈）> 三级信息（媒体、社区、自媒体）。对三级信息要打折扣。
- **信源交叉验证**：单一信息源不可作为决策依据。重大判断必须有两个以上独立来源验证。
- **承认自己不懂**：对于技术路线复杂、商业模式看不清、竞争格局混沌的标的，宁可错过，不做错。

**输出要求**：评估该标的是否在普通投资者能力圈内，并给出信息获取建议。

#### 8.4 行为金融学偏差自检

**灵魂问题**：这个决策是逻辑驱动，还是情绪驱动？

- **常见偏差清单**：
  - 锚定效应：被买入成本绑架，不愿止损。
  - 确认偏误：只收集支持自己观点的信息。
  - 损失厌恶：跌时死扛，涨时过早止盈。
  - 近因效应：被最近涨跌过度影响。
  - 过度自信：把运气当能力。
  - FOMO：害怕错过而追高。
- **自检问题**：如果我现在没有持仓，还会以今天的价格买入吗？如果答案是“不会”，说明你可能被锚定成本影响。

**输出要求**：列出该标的当前最容易触发的 2-3 个行为偏差，并给出纠偏提示。

#### 8.5 退出策略完整性

**灵魂问题**：你计划怎么退出？买入前就必须想好退出。

- **止损退出**：跌破成本 -20% 无条件止损（常总纪律）。
- **技术退出**：跌破 5 日线 7.5% 减仓一半，跌破 20% 清仓（五日战法）。
- **逻辑退出**：时代主线反转、稀缺卡位瓦解、基本面证伪 → 无论盈亏都退出。
- **止盈退出**：估值极度透支、PEG > 1.5、市场狂热、人人都在讨论 → 分批止盈。
- **时间退出**：若 6-12 个月内核心逻辑仍未验证，重新评估是否继续持有。

**输出要求**：为该标的制定明确的退出计划，包括价格、逻辑、时间三类触发条件。

#### 8.6 尾部风险与黑天鹅管理

**灵魂问题**：如果最坏情况发生，你还能活着吗？

- **单一标的上限**：单一个股 ≤ 20%，单一行业 ≤ 30%。
- **相关性检查**：持仓是否过度集中于同一产业链、同一宏观因子？例如 AI 算力中的光模块、PCB、GPU 同时受北美需求影响。
- **流动性风险**：小市值、低成交标的，极端情况下能否顺利退出？
- **政策风险**：行业是否面临监管转向、出口限制、补贴退坡？
- **地缘政治风险**：半导体、军工、新能源、稀土等赛道是否受国际局势影响？

**输出要求**：列出该标的的 2-3 个关键尾部风险，并给出对冲或控制建议。

#### 8.7 交易成本、税费与摩擦

**灵魂问题**：频繁交易是否侵蚀了收益？

- **交易成本**：佣金、印花税、过户费、冲击成本。短线频繁交易可能让年化收益缩水 3-5%。
- **税费**：A 股持股不满 1 个月股息红利税 20%，1 年以上免征；资本利得暂不征税，但政策可能变化。
- **摩擦成本**：大资金的买卖会冲击价格，小盘股尤其明显。

**输出要求**：提醒用户注意交易频率与税费，对短线策略给出摩擦成本提示。

#### 8.8 杠杆与融资纪律

**灵魂问题**：你用了杠杆吗？杠杆会改变一切。

- **绝对纪律**：不建议普通投资者使用融资买入高波动成长股。
- **若已使用融资**：必须同步收紧止损（-10% 甚至 -15%），因为时间不是杠杆的朋友。
- **维保风险**：股价暴跌可能触发追加保证金或强制平仓，导致在底部被动割肉。

**输出要求**：若用户涉及杠杆，给出专门风险提示；若无杠杆，提示保持无杠杆状态。

#### 8.9 产业链交叉验证

**灵魂问题**：公司自己的故事，上下游认不认？

- **上游验证**：原材料供应是否紧张？价格是否上涨？公司能否获得稳定供应？
- **下游验证**：客户订单是否真实？大客户是否扩产？库存是否健康？
- **同行验证**：竞争对手是否也在扩产？行业资本开支是否见顶？
- **海外映射**：全球龙头（如英伟达、台积电、美光）的指引是否支持国内产业链叙事？

**输出要求**：对该标的进行产业链上下游交叉验证，指出验证点与潜在矛盾。

#### 8.10 市场周期状态适配

**灵魂问题**：当前市场处于什么阶段？策略是否需要调整？

- **牛市**：重成长、重赛道、重资金趋势，估值容忍度可适度提高，但需保留止盈纪律。
- **熊市**：重现金流、重估值、重安全边际，轻仓或空仓，等待极端错杀。
- **震荡市**：重结构性机会，重行业轮动，重五日战法快进快出。
- **流动性危机**：所有估值模型失效，现金为王，不抄底、不加仓、先活着。

**输出要求**：判断当前市场周期对该标的策略的影响，给出适配性建议。

#### 8.11 复盘与学习系统

**灵魂问题**：这次决策无论对错，你能从中学到什么？

- **买入前记录**：记录买入理由、预期持有时间、止损位、目标位、最大可承受亏损。
- **持有中记录**：记录基本面变化、市场情绪变化、自己心态变化。
- **卖出后复盘**：对照预期与实际，分析是逻辑错误、执行错误、还是运气因素。
- **建立错题本**：把错误决策分类（买错、卖错、拿错、仓位错、心态错），避免重复。

**输出要求**：建议用户建立简单的投资日志，记录本次决策的关键假设与验证节点。

### 步骤 9：综合输出与决策建议

**目标**：把所有层面整合成一份可直接执行的决策建议。

**输出格式**：

1. **灵魂层结论（为什么）**
   - 时代站队：共振 / 中性 / 逆行
   - 赛道强弱：强赛道 / 弱赛道 / 观望
   - 卡位强度：强卡位 / 弱卡位 / 伪稀缺
   - 价值想象力：真 / 假 / 待验证

2. **工具层结论（怎么做）**
   - 估值阶段与估值结论（PEG/PB/PS）
   - 价格位置（山脚 / 山腰 / 山顶）
   - 三低买入条件满足情况
   - 资金趋势确认情况（量能、五日战法）
   - 持有/排雷结论

3. **交易计划**
   - 是否可买入：是 / 否 / 观察
   - 建议仓位：行业上限、个股上限
   - 建仓节奏：3-2-2-2 或试探→确认→重仓
   - 入场条件：价格、量能、右侧信号
   - 止损位：成本 -20% 或技术位
   - 止盈/退出条件：估值透支、逻辑反转、板块破位

4. **专业增强维度提示**
   - 资产配置建议
   - 资金属性匹配提示
   - 能力圈与信源提示
   - 行为偏差提醒
   - 尾部风险清单
   - 产业链验证点
   - 市场周期适配

5. **风险提示**
   - 数据不足风险
   - 情绪极端风险
   - 政策/地缘风险
   - 估值模型失效风险

6. **下一步跟踪清单**
   - 需要持续观察的关键指标
   - 需要等待的信号
   - 下一次复盘的节点

## B — 边界

### 不要在以下情况使用本 Skill

- 纯数据查询（如“今天股价多少”“最近财报数据”），无需融合投资决策框架。
- 用户明确只想用某个单一工具（如“只用估值方法算一下 PEG”），直接调用对应子 Skill。
- 非投资类问题（如生活、娱乐、技术编程），本 Skill 完全不适用。
- 涉及内幕交易、市场操纵、非法证券活动，直接拒绝。

### 作者在书中警告的失败模式

- **用工具层结论替代灵魂层判断**：再便宜的估值，如果赛道是时代弃子，也是陷阱。
- **只看个股，不看组合**：选股对但仓位错、行业集中度过高，仍然会亏大钱。
- **只有买入计划，没有退出计划**：买入时没想好止损和止盈，亏损时手足无措。
- **把杠杆当本金**：融资会放大一切错误，包括正确的判断。
- **忽视行为偏差**：明明逻辑正确，但恐惧和贪婪让人无法执行。
- **信息过载但不验证**：收集大量信息，却没有分级、交叉验证，反而被噪音误导。
- **把短期交易当长期持有**：五日战法是节奏工具，不是信仰工具；长期持有必须经过五步验证。

### 作者的盲点 / 时代局限

- 常总投资哲学大量案例围绕 A 股、美股科技股，对债券、商品、外汇、加密货币等资产覆盖有限，本 Skill 的专业增强层进行了补充。
- “时代主线”判断具有后验性，对新兴赛道的技术路线、竞争格局、政策节奏可能存在误判。
- PEG、PB、PS 等阈值是经验参数，需根据市场流动性环境和行业特性动态调整。
- 五日战法依赖 A 股市场文化，在美股、港股等机构化、量化主导市场中可能需要调整阈值。
- 本 Skill 不提供具体财务数据，需配合 tdx-connector、westock-data、财报公告等工具获取。

### 容易混淆的邻近方法论

- **与本体系四个子 Skill 的关系**：本 Skill 是总入口，子 Skill 是工具。本 Skill 负责调用和整合，子 Skill 负责专业判断。
- **与一般投资顾问的区别**：本 Skill 强调灵魂层+工具层+专业增强层三位一体，不是简单给出买/卖建议。
- **与纯技术分析的区别**：本 Skill 以时代、赛道、稀缺、估值为底层，技术分析和资金趋势只是执行工具。
- **与纯价值投资的区别**：本 Skill 也看估值和安全边际，但更强调阶段匹配、右侧确认和资金验证。
