# 国策主题投研 Skill

将"昊天意志"（国家战略方向+真金白银投入+不可逆产业趋势）固化为可复用的投资主题研究技能。

## 核心理念

A股所有主线级别的大行情，本质都是顶层政策意志与宏观周期共振的结果。先找到"昊天意志"指向的行业，再在行业内做精选——**先题材后选股**。

> "昊天意志" = 国家战略方向 + 真金白银投入 + 不可逆的产业趋势。三个条件缺一不可。

## 触发条件

当用户问及以下问题或意图时激活：
- "哪些行业符合国家发展方向"
- "昊天意志指向哪里"
- "国家支持的行业有哪些"
- "十五五主线"
- "政策驱动的投资方向"
- "国策主题"
- 任何涉及"先题材后选股"的投资方法讨论

## 工作流程

### 第一步：读取昊天意志（政策源头）

1. 从以下文件中提取明确重点支持的行业清单：
   - 政府工作报告（年度，最新）
   - 十五五规划纲要（2026-2030）
   - 中央经济工作会议通稿
   - 各部委专项政策（工信部、发改委、科技部）
2. 使用 WebSearch/WebFetch 获取最新政策原文
3. 标注每个方向的资金配套（超长期特别国债、产业基金、财政补贴等）

### 第二步：昊天意志行业分层

按政策强度和真金白银投入力度，将行业分为三个层次：

| 层级 | 定义 | 筛选标准 |
|------|------|----------|
| **昊天层** | 国家战略核心，万亿级投入 | 写入政府工作报告核心任务 + 超长期特别国债覆盖 + 举国体制攻关 |
| **天命层** | 明确"新兴支柱产业"，千亿级 | 政府工作报告点名 + 产业基金落地 + 央企国企带头开放场景 |
| **顺风层** | 政策鼓励但非核心，百亿级 | 有政策文件支持但资金规模较小 / 尚在导入期 |

当前（2026年6月）的三层映射：

**昊天层**：
- AI算力（智算集群、算电协同）
- 集成电路（全链条关键核心技术攻关）
- 数据要素（数字经济核心产业占GDP 12.5%）

**天命层**：
- 低空经济（明确"新兴支柱产业"）
- 商业航天（战略前沿领域）
- 新型储能（装机超1.3亿千瓦）
- 生物医药（创新药高质量发展）
- 量子科技（国家实验室部署）

**顺风层**：
- 新能源汽车（渗透率>50%，从增量变存量）
- 具身智能（实验室→产业化过渡）
- 氢能（国家低碳转型基金，导入期）
- 6G（标准制定阶段）

### 第三步：行业→题材映射

每个昊天意志行业拆分为可投资的细分题材和产业链关键环节：

| 行业 | 细分题材 | 产业链稀缺环节 |
|------|----------|---------------|
| **AI算力** | 算力租赁、光模块、PCB、存储、交换芯片 | GPU芯片供给、CoWoS封装、光芯片 |
| **算电协同** | 变压器、HVDC/巴拿马电源、液冷、储能PCS | 干式变压器（全球缺口30%/订单到2027） |
| **集成电路** | 半导体设备、材料、EDA、先进封装 | 光刻机、大硅片、CMP/刻蚀 |
| **低空经济** | eVTOL整机、飞控系统、空管系统 | 适航认证（规则壁垒） |
| **商业航天** | 火箭发射、卫星制造、地面终端 | 发射工位（物理稀缺） |
| **新型储能** | 储能PCS、电池系统、系统集成 | PCS（电力电子核心） |
| **数据要素** | 数据交易平台、高质量数据集、数据安全 | 数据确权/定价机制 |

### 第四步：题材内精选（可选，由用户决定深度）

在选定的题材内，按四个维度筛选：
1. **卡位**——是否卡住产业链不可替代的供给约束
2. **订单**——是否有验证景气度的真实订单数据
3. **利润**——毛利率是否体现定价权
4. **估值**——当前价格是否透支预期

### 第五步：交叉验证

- IMA 知识库【聚宝盆】搜索该题材的历史研究结论
- IMA 知识库【浑水调研】搜索外资研报覆盖（**主力信源**，权重最高）
- IMA 知识库【锅师信息沉淀】仅作辅助对照，不单独作为结论依据
- 通达信/NeoData 拉取最新财务和行情数据
- 产业链瓶颈投研 skill 做上游稀缺层深挖

🔴 **CHECKPOINT** — 交叉验证完成后，对比三层映射与社群/外资/数据端信号是否一致。若出现重大偏离（如昊天层行业无真实订单验证），暂停输出并标注"政策信号与落地信号偏离"。

## 输出规范

🛑 **输出前终审** — 逐条确认：①每层判断有政策出处 ②资金规模有来源 ③时效性标注为"截至XXXX年X月"。任一条件不满足→补充后再输出。

1. **先排层级，再排题材**——不跳步骤，用户应当看到"为什么这个行业受国家支持"的完整逻辑链
2. **不推荐个股**，除非用户明确要求
3. **全程中文**，逻辑推演优先于结论，避免暗语
4. **标注政策来源**——每条判断附上出处（政府工作报告第X章/十五五规划第X条/某部委文件）
5. **注明时效性**——政策环境是动态的，标注"截至XXXX年X月"

## 禁止行为

| # | 禁止项 | 原因 |
|:---|:---|:---|
| 1 | **无政策出处就标注"昊天层"** | 没有政府工作报告/规划原文引用的行业，最多归为顺风层 |
| 2 | **政策支持误读为业绩兑现** | 政策鼓励≠订单落地≠利润增长，三层之间有执行时滞 |
| 3 | **编造不存在的政策原文** | 必须通过 WebSearch/WebFetch 获取最新政策，禁止凭记忆引用 |
| 4 | **输出时遗漏时效性标注** | 政策动态调整，不标"截至XXXX年X月"会误导后续判断 |
| 5 | **跳过第四五六步直接给结论** | 昊天层只是方向判断，必须在第四步（卡位/订单/利润/估值）+ 交叉验证后才算完整投研 |
| 6 | **对"顺风层"做"昊天层"级别的推荐** | 三层之间资源约束和确定性相差一个量级 |

## 风险边界

- 国家政策方向可能因宏观环境变化而调整——保持季度更新
- 政策支持 ≠ 行业一定盈利——执行端、技术端、需求端各有变数
- 本技能提供方向判断和产业链定位，不替代个股基本面研究
- "昊天意志"为分析框架比喻，非官方表述
