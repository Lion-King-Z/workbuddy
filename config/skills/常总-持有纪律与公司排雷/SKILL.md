# 常总-持有纪律与公司排雷

## R — 原文

> 资本市场绝大多数巨额亏损，并非因为没买到牛股，而是因为踩到了“雷”。真正的高手，首要考虑的不是“能赚多少”，而是“会不会踩雷”。长期持有也绝非盲目“买入并遗忘”，而是锁定时代主线、锚定现金流、建立高维认知与资金心智双重容错的一场修行。
>
> — 常士杉，《避开那些“雷”：公司治理风险的识别术》《长期持有的五大黄金准则：穿越周期的认知与修行》

---

## I — 方法论骨架

这个工作流把“能不能长期持有”变成一道可执行的判断题，而不是情绪题。它按顺序完成五件事：先排除公司治理层面的显性地雷；再验证企业是否靠自己赚钱而非靠融资输血；接着判断时代主线是否改变，避免把短期调整当成趋势反转；然后对照长期持有的五条黄金准则，确认标的配得上“时间的朋友”；最后回到投资者自身，检查是不是因为恐惧或焦虑才想交易。任何一步亮起红灯，都不要再往下走，而是先退出或转为观望。只有五步全部通过，才值得继续长期持有。

---

## A1 — 书中的应用

1. **MU（美光）从腰斩到翻倍**：2022年MU从近100美元跌至40多美元，市场一片看空。但AI与HBM存储需求没有改变，时代主线仍在兑现，公司造血能力最终跟上，股价随后大幅回升。这说明主线判断比短期股价波动更重要。

2. **大股东减持与审计异常的“雷”**：某些公司在股价低位或业绩平淡期突然遭大股东高比例减持、更换小所审计、年报延迟披露，往往是公司治理或财务风险暴露的前兆。先离场再观察，是排雷纪律。

3. **英伟达与微软的长期持有**：真正的大钱来自陪伴时代。AI算力、云计算、数据中心是确定性拐点后的颠覆性赛道，叠加持续现金流与定价权，才构成长期持有的底气，而不是简单的“买入并遗忘”。

---

## A2 — 触发场景

1. 已持有某标的，股价下跌或横盘，犹豫要不要卖。
2. 周期股或补涨板块突然大涨， tempted 切换仓位。
3. 标的出现治理、审计、大股东减持等负面消息，担心踩雷。
4. 用户问“这家公司能不能长期持有”“有没有雷”。
5. 账户频繁交易，想验证自己是否该继续拿住。

语言信号：
- “这个票跌了20%要不要卖？”
- “周期股涨了，要不要切过去？”
- “这家公司有没有治理风险？”
- “能不能长期持有？”
- “调整是不是趋势结束了？”

与相邻 skill 区分：
- 与 `常总-赛道稀缺卡位评估` 的区别：那是“准入”判断，决定标的值不值得进入；本 skill 是“持有”判断，决定拿不拿、卖不卖。
- 与 `常总-估值与逆向买入` 的区别：那是“买点”判断，解决何时以什么价格介入；本 skill 是“持有纪律与排雷”，解决拿住与退出。
- 与 `常总-资金趋势识别` 的区别：那是“资金面”跟踪，解决市场短期共识与资金动向；本 skill 聚焦“基本面、治理、时代主线与心态”。

## E — 可执行步骤

当 skill 被激活后，按以下步骤执行。每一步先完成检查，再决定继续或判停。判停即终止流程，给出“不建议长期持有/建议退出或观望”的结论。

### 步骤 1：公司治理排雷

**目标**：识别显性治理与财务风险，宁可错过，不要做错。

**输入**：
- 标的公司近2-3年年报、审计报告、董事会/高管变动公告
- 大股东增减持、股权质押、关联交易、违规担保记录
- 行业负面新闻、监管动态

**检查清单**：
1. **核心人员动荡**：核心技术骨干或关键高管是否任期未满即辞职、是否出现核心团队集体出走。
2. **审计异常**：是否更换会计师事务所（尤其从大所换到小所）、审计意见是否为保留/无法表示/否定意见、年报是否迟迟不发。
3. **大股东异常行为**：是否股价低位或业绩平淡期大幅减持、是否高比例质押、是否占用上市公司资金或违规担保。
4. **资本运作频繁**：是否频繁高溢价跨行业并购、是否频繁改名蹭热点、是否高送转配合减持。
5. **行业共振**：是否同行批量暴雷、监管是否集体约谈、行业政策是否从鼓励转向严管。

**完成标准**：
- 五项检查均无红灯；或虽有黄色信号，但已充分解释且不影响长期持有逻辑。

**判停条件**：
- 任一红灯亮起（核心人员集体出走、审计意见非标、大股东资金占用、频繁蹭概念并购、行业政策转向严管）。
- 出现红灯后，不再进入步骤2，直接给出“存在治理/财务/行业风险，建议先退出或转为观望”的结论。

**输出**：
- 治理风险清单：绿灯/黄灯/红灯
- 若红灯，输出：退出建议 + 风险类型

---

### 步骤 2：自主造血验证

**目标**：确认企业能否靠自己赚钱，而非依赖外部融资输血。

**输入**：
- 近5年（至少3年）利润表、现金流量表
- 资本开支、自由现金流（FCF）、经营现金流净额
- 毛利率、净利率、期间费用率变化

**检查清单**：
1. **自由现金流（FCF）是否持续为正**：关注扣除非经常性损益后的真实FCF，是否逐年改善或稳定。
2. **经营现金流是否稳定**：经营现金流是否覆盖资本开支，是否出现大幅波动或持续低于净利润。
3. **毛利率结构是否具备定价权**：毛利率是否稳定或提升，是否依赖价格战、补贴、一次性订单；提价后销量是否保持。
4. **增长质量**：收入增长是否伴随现金流同步增长，还是“增收不增现”。

**完成标准**：
- FCF持续为正或趋势向好；经营现金流稳定覆盖资本开支；毛利率体现定价权或护城河；增长不是单纯资本驱动。

**判停条件**：
- FCF长期为负且无明显改善；经营现金流无法覆盖资本开支；毛利率持续下滑且无定价权；增长严重依赖融资输血。
- 任一判停条件满足，输出“造血能力不足，不宜长期持有，可列为交易性机会或剔除”。

**输出**：
- 自主造血评估：健康/亚健康/不可持续
- 关键指标表格与趋势说明

---

### 步骤 3：主线坚守判断

**目标**：区分“时代主线的短期调整”与“趋势反转”，避免因为补涨轮动而离开主线。

**输入**：
- 标的所属行业的产业趋势、渗透率、技术路线、政策与资本动向
- 相关上下游数据（订单、资本开支、库存、价格）
- 宏观政策与资金环境变化

**检查清单**：
1. **时代级趋势是否改变**：AI、机器人、云计算、新能源、数据中心等长期趋势是否仍然成立，是否只是“涨多了需要降温换手”。
2. **政策与资本是否继续支持**：产业政策是否从支持转向压制，资本开支是否仍在流入该领域，是否有新的应用场景出现。
3. **产业是否仍在兑现**：龙头订单、营收、产能利用率、技术迭代是否继续验证，而非仅停留在概念。
4. **区分调整与反转**：
   - 调整信号：涨多了、拥挤度高、短期获利盘了结、估值偏高，但基本面未变。
   - 反转信号：技术路线被替代、需求拐点下行、政策转向、核心客户流失、行业资本开支见顶。

**完成标准**：
- 时代级趋势未改变，政策与资本继续支持，产业仍在兑现；当前下跌更符合“调整”而非“反转”。

**判停条件**：
- 确认趋势反转（技术路线被颠覆、需求系统性下行、政策由支持转严管、行业资本开支见顶回落）。
- 反转确认后，即使浮亏，也应输出“趋势反转，建议退出，不恋战”。

**输出**：
- 主线判断：调整 / 反转 / 不确定（不确定时要求降低仓位或继续观察）
- 关键证据与逻辑链

---

### 步骤 4：长期持有准则验证

**目标**：对照五条黄金准则，判断标的是否真的配得上“长期持有”。

**输入**：
- 赛道属性与技术/商业壁垒
- 产业发展阶段与渗透率数据
- 业绩、现金流、盈利能力验证
- 投资者自身对该赛道的认知深度与资金周期

**检查清单**：
1. **是否锁定颠覆性赛道**：赛道是否具备重构产业格局或改变人类文明的潜力，是否拥有跨越周期的内生增长动力。
2. **是否识别确定性拐点**：是否出现技术突破到商业化落地的明确信号，渗透率是否突破临界点，营收是否进入指数级爆发阶段。
3. **是否有资金与心智双重容错**：投资资金是否具备超长周期属性；投资者是否能承受50%甚至70%以上回撤而不被情绪左右。
4. **是否锚定业绩与现金流**：长期叙事是否已兑现为真金白银的利润，自由现金流是否持续增长，技术优势是否转化为定价权。
5. **是否具备高维认知**：投资者对赛道的技术演进、商业逻辑是否具备远超市场多数参与者的理解，能否解释“为什么是它”。

**完成标准**：
- 五项准则全部满足或至少满足“颠覆性赛道+确定性拐点+业绩现金流+高维认知”，且资金周期匹配。

**判停条件**：
- 任一准则明显缺失：例如赛道只是概念、拐点未现、无法承受回撤、业绩现金流无法验证、认知仅停留在消息面。
- 缺失项输出“不符合长期持有条件，建议降低仓位或转为波段交易”。

**输出**：
- 长期持有适配度：适配 / 部分适配 / 不适配
- 五准则逐项打分与结论

---

### 步骤 5：无为心态检查

**目标**：确保交易决策不是由恐惧、焦虑或FOMO驱动，而是基于“自己的机会”。

**输入**：
- 投资者近期交易记录、换手率
- 当前持仓比例与可用资金
- 投资者自述的心理状态（可用问卷或对话获取）

**检查清单**：
1. **是否因为恐惧/焦虑而交易**：下跌是否想割肉、上涨是否想追、横盘是否想换仓，是否因害怕错过而频繁操作。
2. **是否只在“自己的机会”上动手**：是否清楚自己等待的关键信号，是否只在认知范围内的绝对关键点行动。
3. **是否保留战略预备队**：账户是否保留了足够现金/仓位，以便在真正的好机会或极端低估时加仓；是否把子弹打光。
4. **是否把账户当作“判断力说明书”而非“行动记录本”**：80%的时间是否用于蛰伏观察，20%的时间用于重拳出击。

**完成标准**：
- 交易动机清晰、非情绪驱动；明确知道自己的关键信号；保留战略预备队；愿意在无效波动中保持不动。

**判停条件**：
- 确认情绪驱动交易（因恐惧、FOMO、控制幻觉、行动成瘾而频繁操作）。
- 判停后输出“先停止交易，回到观察状态，等待自己的机会，而不是被市场情绪牵引”。

**输出**：
- 心态状态：稳定 / 情绪驱动 / 需要重置
- 行动建议：继续持有 / 暂停交易 / 降低仓位 / 只观察不操作

---

### 流程终点与综合输出

**若五步全部通过**：
- 结论：标的具备长期持有的基本条件，投资者心态稳定。
- 建议：继续持有；如基本面进一步强化且出现好价格，可加仓；但需保留战略预备队。

**若任何一步判停**：
- 结论：当前不适合长期持有。
- 建议：根据判停步骤给出具体行动——退出、减仓、观望、暂停交易或转为波段机会。

**持续跟踪**：
- 每季度复检步骤1（治理）、步骤2（造血）、步骤3（主线）。
- 每半年或重大波动后复检步骤4（长期持有准则）与步骤5（心态）。

## B — 边界

### 不要在以下情况使用此 skill
- 用于判断某只股票是否值得“买入”（应使用 `常总-赛道稀缺卡位评估` 或 `常总-估值与逆向买入`）。
- 用于判断短期资金趋势或热点轮动（应使用 `常总-资金趋势识别`）。
- 用于决定杠杆、融资、仓位比例等资金配置问题（本 skill 只判断“能不能拿”，不直接给出仓位）。
- 用于判断加密货币、期货、外汇等非股票资产。

### 作者在书中警告的失败模式
- 把核心人员动荡、高管集体出走当成“正常人事变动”。
- 把审计异常、非标意见当成“会计师事务所太保守”。
- 把大股东低位减持、高比例质押当成“个人资金需求”。
- 把频繁跨行业并购、改名蹭热点当成“公司有战略眼光”。
- 把行业政策转向严管当成“短期利空，很快修复”。
- 把“收入高速增长”等同于“好公司”，忽视现金流与造血能力。
- 把时代主线的短期调整当成“趋势结束”，追涨补涨板块。
- 把频繁交易当成“勤奋”，实则是情绪调节与本金消耗。

### 作者的盲点 / 时代局限
- 常士杉的案例以美股科技股、A股白马为主，对高杠杆、强周期、政策干预剧烈的行业（如房地产、金融、部分资源股）适用性需打折。
- A股信息披露与退市机制相对特殊，部分治理风险信号可能滞后或隐蔽，需结合更多本地监管公告与舆情交叉验证。
- “时代主线”判断高度依赖事后归纳，对新兴赛道的技术路线、竞争格局、政策节奏可能存在误判。
- 个人投资者的资金周期、风险承受能力、现金流需求差异很大，五条准则中的“双重容错”需要结合用户自身情况调整，而非机械套用。
- 本 skill 不涉及个股财务建模与估值，需配合其他 skill 或工具获取具体财务数据。

### 容易混淆的邻近方法论
- 与止损/止盈交易纪律混淆：止损是价格与风险管理的交易规则，本 skill 是基本面与时代主线驱动的持有/退出判断。
- 与价值投资“安全边际”混淆：安全边际解决“买得便宜”，本 skill 解决“拿得安稳”。
- 与宏观经济择时混淆：宏观择时试图判断市场顶底，本 skill 只判断标的是否仍值得陪伴。
- 与“死多头/死空头”混淆：长期持有不是无条件持有，而是经过五步验证后的有条件持有；红灯出现必须退出。
