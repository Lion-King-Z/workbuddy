# 常总-赛道稀缺卡位评估

## R — 原文

> 所有超级大牛股，本质都不是“公司牛”，而是赛道牛。你以为你在投资一家公司，其实你是在押注一个时代。
>
> — 常士杉，《私募一哥常士杉·投资哲学系列①——超级赛道哲学》

> 普通投资者：找“好公司”。顶级投资者：找“不可替代的公司”。真正的大牛股，不是最优秀的，而是唯一的。
>
> — 常士杉，《超级成长投资哲学系列④：行业稀缺——垄断与卡位，才是真正的护城河》

> 资本市场从来不是独立系统，它只是文明运行的一部分。真正决定财富命运的力量，来自更宏大的维度：国家战略、科技革命、货币体系、战争格局。
>
> — 常士杉，《私募一哥常士杉·星球专供〈国运投资学〉》

---

## I — 方法论骨架

这个工作流把“选赛道”和“选公司”两件事串成一个连续判断流程：先看时代站在哪一边，再看国家、资本、产业是否在同一个方向共振，最后看目标公司是否卡在产业链中别人替代不了的位置。它不是独立分析一堆指标，而是按“国运→赛道→卡位”顺序逐层过滤，每一层都有判停条件，通不过就直接出局或降级，不进入下一步。

---

## A1 — 书中的应用

### 案例1：AI算力是否值得长期押注

- **问题**：AI算力是不是长期超级赛道？
- **方法使用**：时代框架判断（AI为新一轮科技革命）→ 政策方向（国家战略扶持数字经济与算力基建）→ 资本流向（AI ETF、算力龙头持续放量）→ 产业验证（英伟达、国产AI芯片商业化落地）→ 稀缺卡位（高端GPU/先进封装/HBM材料）。
- **结论**：强赛道，具备牌照、技术、产业链三重稀缺卡位。
- **结果**：赛道方向确认，下一步进入估值与买入时机判断。

### 案例2：HBM材料公司是否具备稀缺卡位

- **问题**：某HBM材料公司值不值得投？
- **方法使用**：赛道确认为AI算力强赛道后，评估稀缺来源。HBM材料处于上游核心原材料节点，技术壁垒高、认证周期长、替代成本高；若公司市占率持续提升且下游客户绑定头部存储厂，则具备产业链卡位+技术壁垒。
- **结论**：强赛道强卡位。
- **结果**：纳入核心观察池，等待估值与资金信号配合。

### 案例3：夕阳行业中的“优质公司”

- **问题**：某传统制造公司PE很低、财务稳健，是否值得长期持有？
- **方法使用**：先做时代框架判断——该行业不在国家战略、科技革命、资本流向的任何主线上；再做超级赛道识别——政策无扶持、资本持续流出、产业无新龙头跑出。
- **结论**：弱赛道，直接判停。
- **结果**：不进入稀缺卡位评估，建议放弃或仅作短期套利。

## A2 — 触发场景

1. 用户问某个行业/赛道是否具备长期投资价值。
2. 用户问某家公司有没有护城河、是不是稀缺卡位标的。
3. 用户在多个赛道/公司之间做比较，需要判断“方向对不对、位置卡没卡准”。

语言信号：
- “AI算力是不是长期超级赛道”
- “HBM材料有没有稀缺卡位”
- “这个方向值不值得投”
- “这家公司有没有护城河”
- “是不是不可替代的公司”

与相邻 skill 区分：
- 与 `常总-估值与逆向买入` 的区别：本 skill 回答“赛道对不对、卡位强不强”，估值 skill 回答“现在价格贵不贵、能不能买在低估区”。
- 与 `常总-资金趋势识别` 的区别：本 skill 回答长期方向与产业地位，资金趋势 skill 回答短期/中期资金是否进场、趋势是否成立。
- 与 `常总-持有纪律与公司排雷` 的区别：本 skill 用于买入前的方向筛选，持有排雷 skill 用于持有过程中的风险监控与退出。

## E — 可执行步骤

当 skill 被激活后，按以下步骤执行：

### 步骤1：国运/时代框架判断

判断当前标的所在的宏观方向是否站在文明周期的主线上。

- **完成标准**：能够回答以下四个问题，并给出“时代共振 / 时代中性 / 时代逆行者”三档判定：
  1. **国家战略**：该方向是否被国家战略明确扶持？（政策文件、五年规划、重大专项、财政补贴）
  2. **科技革命**：是否属于新一轮科技革命的核心基础设施？（AI、算力、数据、新能源、生物科技、航空航天等）
  3. **货币体系**：当前全球/国内货币环境是否有利于该方向的估值扩张？（宽松周期利于成长资产，紧缩周期压缩估值）
  4. **战争格局**：地缘冲突是否加速该方向的需求？（军工、无人机、卫星互联网、网络安全、能源安全、半导体自主可控）

- **输入**：标的所属行业、国家相关政策、全球宏观环境、科技产业趋势。
- **输出**：时代框架判定（共振/中性/逆行），并列出支撑与否定证据。
- **判停条件**：
  - 若判定为“时代逆行者”：直接判停，输出“弱赛道/不建议”，不进入步骤2。
  - 若判定为“时代共振”或“时代中性”：进入步骤2。

### 步骤2：超级赛道识别

在时代框架通过的基础上，用政策、资本、产业三维度验证该赛道是否具备“长周期、万亿级、不可逆”的特征。

- **完成标准**：三个维度至少满足两项，且“产业验证”必须为正：
  1. **政策方向**（钱往哪里流，而不是看说什么）：
     - 是否有持续财政补贴、税收优惠、产业基金投入？
     - 是否写入国家级战略规划？
  2. **资本流向**（聪明钱是否已经下注）：
     - 相关ETF/主题基金是否持续净流入？
     - 赛道龙头是否走出趋势性新高或持续放量？
     - 一级市场融资是否活跃？
  3. **产业验证**（必须有，不可缺）：
     - 是否有龙头公司已经跑出来？
     - 是否进入规模化商业化或“爆发前夜”？
     - 产业链是否形成闭环？

- **输入**：政策支持清单、资金流向数据、产业链成熟度信息、龙头公司动态。
- **输出**：超级赛道判定（强赛道 / 弱赛道 / 观望）。
- **判停条件**：
  - 若“产业验证”为负：判停，输出“观望/等待产业验证”，不进入步骤3。
  - 若三维度仅满足一项或零项：判停，输出“弱赛道”。
  - 若满足两项及以上且产业验证为正：进入步骤3。

### 步骤3：稀缺卡位评估

确认赛道之后，评估目标公司在赛道中的不可替代性。

- **完成标准**：识别至少一种稀缺来源，并判断其在产业链中的卡位层次。

  **稀缺来源五选一（至少命中一项）**：
  1. **牌照垄断**：金融、数字货币、医疗审批、军工资质等政策性准入壁垒。
  2. **技术壁垒**：核心专利、独家工艺、算法、Know-how，复制成本极高。
  3. **资源垄断**：矿产、能源、稀土、数据等上游核心生产要素控制权。
  4. **平台效应**：社交网络、交易平台、生态系统的正反馈与网络效应。
  5. **产业链卡位**：核心零部件、上游关键节点，产品价值占比小但替代成本高。

  **卡位层次三选一（明确所处位置）**：
  1. **上游控制**：原材料、核心资源，控制产业链成本。
  2. **中游核心**：核心技术、核心组件，控制产品质量与性能。
  3. **下游入口**：用户、流量、数据，控制市场需求。

  **同时必须满足稀缺有效性三条件**：
  - 有稀缺：具备五大来源之一。
  - 有需求：市场对其产品/服务有真实且持续的需求。
  - 有利润：稀缺性能够转化为收入和利润。

- **输入**：目标公司业务结构、竞争格局、市占率变化、进入壁垒、财务数据。
- **输出**：稀缺卡位判定（强卡位 / 弱卡位 / 伪稀缺），并说明稀缺来源与卡位层次。
- **判停条件**：
  - 若“有需求”或“有利润”为负：判停，输出“伪稀缺/投资陷阱”，不进入步骤4。
  - 若命中稀缺来源且卡位层次清晰：进入步骤4。

### 步骤4：竞争格局与定价权验证

在稀缺卡位基础上，验证公司是否正在形成垄断趋势和定价权。

- **完成标准**：回答以下三个问题：
  1. **市占率是否在持续提升？**（竞争格局优化）
  2. **是否形成垄断趋势或寡头格局？**（定价权萌芽或确立）
  3. **新进入者是否困难？**（进入壁垒高，短期难以被颠覆）

- **输入**：行业市占率数据、毛利率/净利率变化、新进入者动态、客户转换成本分析。
- **输出**：竞争格局判定（垄断趋势确立 / 竞争中 / 红海）。
- **判停条件**：
  - 若三项均为负面：判停，输出“强赛道弱卡位/竞争激烈”。
  - 若至少两项为正：进入步骤5。

### 步骤5：综合判定

整合前述四层判断，输出最终结论与主要风险。

- **完成标准**：给出以下四类结论之一，并列出2-3个核心风险：

  | 结论 | 定义 | 操作建议 |
  |---|---|---|
  | **强赛道强卡位** | 时代共振 + 超级赛道 + 强稀缺卡位 + 竞争格局优化 | 纳入核心池，下一步进入估值与买入时机判断 |
  | **强赛道弱卡位** | 时代共振 + 超级赛道，但公司缺乏不可替代性 | 观察赛道内更优卡位标的，或等待公司卡位提升 |
  | **弱赛道** | 时代逆行或赛道未通过三维度验证 | 放弃，不进入下一步 |
  | **观望** | 方向对但产业验证不足，或稀缺性未转化为利润 | 持续跟踪，等待信号确认 |

- **输入**：步骤1-4的全部判定结果。
- **输出**：最终结论、主要风险、下一步建议。
- **判停条件**：此步骤为终点，输出结论后 skill 结束。

## B — 边界

### 不要在以下情况使用此 skill
- 用户只问某只股票的短期买卖点或技术面信号。
- 用户要求进行详细财务估值建模（PE/PB/DCF等）。
- 用户只想了解某个行业的基本信息或新闻动态，不做投资判断。
- 用户问的是已持有标的的卖出/止损/排雷决策。

### 作者在书中警告的失败模式
- ❌ 只看公司，不看行业 → 选到“夕阳行业的优质公司”。
- ❌ 只看估值，不看趋势 → 买便宜的垃圾。
- ❌ 只看短期，不看长期 → 永远赚不到大钱。
- ❌ 把“稀缺”与“一定赚钱”画等号 → 技术很强但没人用，是“伪稀缺”陷阱。
- ❌ 忽视货币体系与战争格局 → 在错误的时代周期重仓错误资产。

### 作者的盲点 / 时代局限
- 原文大量案例围绕AI、算力、半导体、新能源等当下热点，对消费、医疗、金融等传统超级赛道的讨论相对较少。
- “国家战略”视角主要基于中国政策语境，对海外市场投资需额外补充当地政策与资本环境分析。
- 对“战争格局”的论述偏宏观，未深入讨论具体冲突的阶段性影响与退出信号。

### 容易混淆的邻近方法论
- 与“行业景气度分析”的区别：景气度看短期供需与盈利周期，本工作流看长期时代方向与不可替代性。
- 与“护城河分析”的区别：传统护城河强调品牌、成本、规模，本工作流强调“稀缺来源”与“产业链卡位”。
- 与“主题投资”的区别：主题投资追逐概念与情绪，本工作流要求政策、资本、产业三重验证，并必须落地到稀缺卡位。
