# 个股风险扫描器 Skill v2.1

**定位**：A股个股**多维风险画像生成器** + 原「两融风控师」全部能力的集成升级。

**v2.1 核心升级**（2026-07-18）：
1. 🆕 从14维风险扩展为**15维风险扫描**——新增「资金流向风险」（北向/龙虎榜/大宗交易）
2. 🆕 输出标准「风险画像」——每个维度给出风险等级、标签、对估值/持仓的影响
3. 🆕 保留并强化原两融风控模块：杠杆、维保、爆仓价位、踩踏烈度、情绪温度计

## 四、风险维度详细评估

### 4.1 杠杆风险（原两融风控专业模块）

保留原两融风控师全部计算逻辑：
- 简化 1:1 杠杆假设：P_T = T × C / 2
- P_150 = 0.75 × C；P_130 = 0.65 × C；P_115 = 0.575 × C
- C_weighted = Σ(融资买入额_i × 当日均价_i) / Σ(融资买入额_i)
- C_high：近3日最高成本入场价
- F_ratio = 融资余额 / 流通市值 × 100%
- 踩踏烈度：<3%低 / 3-8%中 / >8%高
- 情绪温度计：历史分位 + 斜率 + 背离 + 主动/被动去杠杆

### 4.2 质押风险

**关键指标**：
- 大股东质押比例 = 大股东质押股数 / 其持股总数
- 质押融资余额 / 总市值
- 预警线/平仓线（通常160%/140%，具体看公告）
- 股价距平仓线跌幅

**风险等级**：

| 大股东质押比例 | 风险等级 | 标签 |
|:---:|:---:|:---|
| <30% | 🟢 低 | 质押风险可控 |
| 30%-50% | 🟡 中 | 质押比例偏高，需关注股价波动 |
| 50%-70% | 🟠 较高 | 高质押，股价大跌可能触发平仓 |
| >70% | 🔴 高 | 极高质押，控制权变更风险 |

### 4.3 信用风险

从财报中提取：
- 资产负债率
- 有息负债/总负债
- 流动比率 / 速动比率
- 利息覆盖倍数
- 对外担保金额 / 净资产

**风险等级**：

| 综合判断 | 标签 |
|:---|:---|
| 负债率低、短债少、利息覆盖高 | 🟢 信用风险低 |
| 资产负债率50-70%、短债占比适中 | 🟡 信用风险中等 |
| 资产负债率>70%、短债占比高、利息覆盖<3 | 🔴 信用风险高 |

### 4.4 商誉风险

**关键指标**：
- 商誉 / 净资产
- 商誉 / 净利润
- 业绩承诺完成率（并购标的）
- 是否已计提减值

**风险等级**：

| 商誉/净资产 | 风险等级 | 标签 |
|:---:|:---:|:---|
| <10% | 🟢 低 | 商誉风险可控 |
| 10%-30% | 🟡 中 | 商誉占比偏高，需关注并购标的业绩 |
| 30%-50% | 🟠 较高 | 高商誉，减值风险大 |
| >50% | 🔴 高 | 极高商誉，业绩承诺不达预期可能大额减值 |

### 4.5 客户集中度风险

| 前五大客户占比 | 风险等级 | 标签 |
|:---:|:---:|:---|
| <30% | 🟢 低 | 客户分散 |

## 七、与每日窄门筛股的集成

每日窄门筛股v4.9对买入建议前5名强制调用@skill:"个股风险扫描器"：

1. 调用输入：股票代码 + 诊断日期
2. 必须获取：
   - 综合风险评级
   - 风险扣分
   - 核心风险标签（最多5个）
   - 两融状态（安全/警戒/追保区/强平区）
   - P_130 / P_115
   - 融资余额/流通市值
   - 踩踏烈度
   - 情绪温度
3. 窄门筛股引用：
   - 在「风险聚合矩阵」中呈现风险标签
   - 在「前5名两融价格区间对比」中呈现两融数据
   - 在买入建议表风险提示中引用核心风险标签
   - 根据风险扣分做风险调整评级

**禁止**：窄门筛股自行计算风险指标，必须引用个股风险扫描器输出。

## 八、注意事项与边界

- 本 Skill **不提供买入/卖出/加仓/减仓等具体操作指令**，只输出风险状态与画像。
- 14维风险中，部分维度（如客户集中度、供应商集中度）年报才披露，季度数据可能缺失。缺失时标注"数据缺失"，不强行判断。
- 风险等级基于公开数据，不代表真实风险全貌。重大未公开风险无法识别。
- 温度计模式使用生活化比喻，禁用"维保比例""折算率""级联强平"等术语，必要时用括号浅释。
- 必须区分「主动去杠杆」和「被动去杠杆」，这是情绪温度计的核心价值。
- 个股必须放在行业+大盘两层背景下解读，不可孤立判断。
- 若无法获取两融数据，明确告知缺失项，并提供替代指标（如换手率、龙虎榜资金）。
- 若用户要求非A股（港股、美股）的风险扫描，说明本 Skill 主要适用于 A 股，部分维度（如两融、质押规则）不适用。
- 严格遵循决策边界铁律：AI 只排序、评级、标注风险；不淘汰、不替用户做主。
- 不得承诺任何收益或回撤幅度，风险声明必须保留。
