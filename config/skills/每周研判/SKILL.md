# 每周研判 Skill

从浑水跟踪月度总账 + 聚宝盆索引中提取过去一周的变化，生成结构化周度报告。

## 核心逻辑

三件事：
1. 浑水跟踪了哪些信号 → 本周变化了什么
2. 聚宝盆新增了哪些研究 → 本周产出了什么
3. 两者交叉 → 什么信号在强化、什么在消退

## 触发条件

- 用户说「本周研判」「周度总结」「一周信号变化」「每周报告」「周报」
- 建议配置自动化：每周日晚 21:00（RRULE: `FREQ=WEEKLY;BYDAY=SU;BYHOUR=21`）

## 工作流

### 第一步：读取信号源

**1A. 读取浑水跟踪月度总账**
- 文件路径：`F:\workbuddy\锅师\浑水调研\浑水跟踪_YYYY-MM.md`（公司机）或 `G:\锅师\浑水调研\浑水跟踪_YYYY-MM.md`（家用机）
- 提取内容：累计信号强度表 + S级信号记录表
- 读取方法：Read 工具直接读取本地文件

**1B. 读取聚宝盆索引**
- 文件路径：`F:\workbuddy\锅师\聚宝盆索引.md`（公司机）或 `G:\锅师\聚宝盆索引.md`（家用机）
- 提取内容：最近7天的新增研究记录（按日期筛选）
- 读取方法：Read 工具读取全文 → 按日期过滤本周条目

**1C. 读取IMA聚宝盆知识库（可选，深度引用用）**
- 知识库 ID：`9P9LfmFJRFwLidUrzKQu0cZZ5QMVxtv61QA9iFoV90g=`
- 提取内容：本周新增的电话会分析笔记
- 读取方法：IMA MCP `get_knowledge_list` 按 `UPDATE_TS_DESC_SORT_TYPE` 排序，limit=20

**1D. 时间窗口确认**
- 本周时间窗口：本周一 00:00 ~ 本周日 23:59
- 周一计算：`当前日期 - timedelta(days=当前日期.weekday())`
- 周日计算：`周一 + timedelta(days=6)`

🔴 **CHECKPOINT — 第一步出口**：确认以下条件全部满足再进入第二步：
- [ ] 浑水跟踪月度总账已读取（当月文件存在且非空）
- [ ] 聚宝盆索引已读取（文件存在）
- [ ] 时间窗口已明确（本周一00:00 ~ 本周日23:59）
- [ ] IMA本周新增笔记已拉取（或标注「IMA不可达，跳过深度引用」）

🛑 **STOP**：若浑水跟踪月度总账不存在 → 提示用户「本月尚未开始电话会跟踪，无周报可生成」并终止。

### 第二步：信号变化分析

| 分析维度 | 方法 |
|----------|------|
| **信号新增** | 本周首次出现的行业/环节 |
| **信号强化** | 提及频次上升、紧缺度升级的环节 |
| **信号消退** | 上周有本周无，或紧缺度降级 |
| **S级信号状态** | 新触发 / 持续活跃 / 已消退 |

🔴 **CHECKPOINT — 第二步出口**：每个变化维度至少有1条记录或明确标注「本周无变化」。

### 第三步：研究产出统计

| 统计项 | 来源 |
|--------|------|
| 本周产业链研究 | 索引中「产业链」类型的新增条目 |
| 本周电话会分析 | 索引中「电话会」类型的新增条目 |
| 本周个股研究/专题 | 索引中「个股」「专题」类型的新增 |

### 第四步：交叉验证

电话会信号 vs 产业链研究产出 → 有无背离？
- 电话会多次提及但未做深度研究 → 标注「待研究」
- 做了深度研究但电话会未提及 → 标注「独立发现」

🔴 **CHECKPOINT — 第四步出口**：每个「待研究」和「独立发现」标注必须附1句话理由。

### 第五步：输出周报

同步到 IMA 聚宝盆（深度研究/当前周文件夹）+ 本地 `深度研究/<周文件夹>/YYYYWww_每周研判.md`

🔴 **CHECKPOINT — 第五步出口**：确认三件套全部完成：
- [ ] IMA 聚宝盆笔记已创建（folder_id=当前周文件夹）
- [ ] 本地 md 文件已写入正确路径
- [ ] 聚宝盆索引已追加周报记录

## 异常与边界条件

| 触发条件 | 一线修复 | 仍失败兜底 |
|----------|----------|-----------|
| 浑水跟踪月度总账不存在 | 检查上月文件是否延续 | 🛑 终止，提示「本月无跟踪数据」 |
| 聚宝盆索引.md 不存在 | 从 IMA 知识库重建索引 | 标注「索引缺失，信号统计不完整」 |
| 本周零信号零产出 | 仍输出周报，所有表格标「本周无变化」 | — |
| IMA 聚宝盆 API 不可达 | 仅输出本地 md | 标注「IMA同步失败，待手动补传」 |
| 周文件夹计算错误（跨月） | 按产出日期归入当周，跨月不拆分 | — |
| 信号源冲突（浑水跟踪与聚宝盆索引不一致） | 以浑水跟踪为准（电话会原始记录） | 标注冲突点 |
| 紧缺度排名与S级信号矛盾 | 优先采信S级信号（已通过多重验证） | 标注分歧 |

## 禁止行为

- 跳过信号源读取直接输出
- 遗漏本周任何一条电话会信号
- 输出"数据源："、"电话会N份"等元数据说明行（对阅读无价值，直接入正文）
- 输出"本周研究产出"统计表（冗余信息，核心是紧缺排名+标的）
- 周报生成后不沉淀到IMA聚宝盆
