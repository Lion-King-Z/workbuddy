# 每周研判 Skill v3.2

**定位演进**：v1.0（汇总排名）→ v2.0（拐点识别+操作建议）→ v3.0（+动态宏观噪音过滤+主题SWOT+风险收益比）→ v3.1（+宏观权重分配器+供给刚性分类映射）→ v3.2（+同步三因子框架：B/G/催化剂）

**v3.2 核心升级**：同步锅师三因子框架（B/G/催化剂），移除E（证据累积）独立引用。证据累积不再作为独立维度——它是催化剂链上的验证状态，不是评分项。

## 核心逻辑

不做排名，做五件事：
1. 拐点识别 — 本周信号方向变化：加速恶化 / 观察升级确认 / 预期差最大
2. **动态宏观噪音过滤器**（v3.0新增）— 不是固定指标清单，而是回答「当前什么在影响钱往哪流？」
3. 交叉验证 — 电话会主观判断 vs 紧缺度日报数据，有无背离
4. **主题SWOT + 风险收益比**（v3.0新增）— 主题级优劣势/机会/威胁 + 中长期vs短期赔率
5. 操作建议 — 基于以上结论，输出可执行的加仓/减仓/提问方向

## 锅师投资逻辑链（v3.2 总纲）

```
Step 1: 宏观噪音过滤 → "当前什么在影响钱往哪流？"
         └─ 动态权重，每周刷新，区分噪音(≤4周) vs 结构性(>3月)

Step 2: 产业链卡点定位 → "哪个环节供给最刚性？"
         └─ Chokepoint Hunter 六步法，两步漏斗 + 供给刚性分类

Step 3: 信号评分 → "三因子加权：瓶颈(B)×预期差(G)×催化剂链"
         └─ 窄门筛股，C为质量乘数，E已消亡
         └─ 仓位=赔率(B×G)×胜率(催化剂兑现率)×确定度(C乘数)

Step 4: 催化剂时机 → "什么时候会被市场发现？"
         └─ 催化剂链三层递进(近/中/远)+景气度阶段

Step 5: 主题SWOT + 风险收益比 → "中长期赔率 vs 短期风险"
         └─ 每周研判输出，动态调整
```

每周研判覆盖 Step 1 和 Step 5 的周度刷新，是整个体系的「方向舵」。

## 工作流

### Step 0：聚宝盆缓存引用

搜聚宝盆（KB: `9P9LfmFJRFwLidUrzKQu0cZZ5QMVxtv61QA9iFoV90g=`）本周全部电话会 + 紧缺度日报 + 窄门筛股输出标题。
提取已有判断→引用不重算。日度已分析过的信号→直接引用结论→只做「本周状态更新」。
目的：周报是信号聚合+状态更新+方向判断+宏观过滤，不重新分析。

### Step 1：拐点识别（核心）

读取浑水调研/浑水跟踪_YYYY-MM.md + 本周紧缺度日报，回答三个问题：

**Q1：哪些信号在加速恶化？**
- 本周紧缺度5/5维持未缓解 + 涨价幅度继续扩大 → 恶化方向
- 本周紧缺度从3/5→4/5或4/5→5/5 → 加速恶化
- 输出：恶化方向清单，按紧迫性排序，每个带核心数据和来源引用

**Q2：哪些信号从「观察」升级到「确认」？**
- 电话会S级信号有新的公开验证 → 升级确认
- 正文NLP提取累积一周达到确认阈值 → 升级确认
- 输出：升级信号清单 + 可操作标的

**Q3：哪个方向是本周最大预期差？**
- 市场关注度低 + 实际紧缺度≥4/5 → 正预期差（市场没发现的机会）
- 市场关注度高 + 紧缺度实际在缓解 → 负预期差（过度拥挤的风险）
- 输出：预期差 Top 3

### Step 1.5：动态宏观噪音过滤器（v3.0新增）

**核心理念**：不设固定宏观指标清单。每周回答一个核心问题——「当前什么在影响钱往哪流？」

四个维度扫描，每个维度识别当前活跃的因子（≤5个/维度）：

| 维度 | 核心问题 | 当期活跃因子示例（2026/06/21） | 数据来源 |
|:---|:---|:---|:---|
| **资金流** | 钱在流向哪/流出哪？ | SpaceX IPO吸金/回购对冲、SOX反弹收复、北向资金方向 | WebSearch + tdx-connector |
| **情绪极值** | 恐慌还是贪婪？ | VIX位置、SOX单日波动幅度、散户热帖关键词 | WebSearch + 雪球情绪 |
| **政策/事件不确定性** | 什么即将发生但结果未知？ | FOMC后鹰派风险、中美关税窗口、世界杯影响 | FOMC官方时间表 + WebSearch |
| **跨市场联动** | 外部冲击是否传导？ | 美债10Y位置、USD/CNY汇率、WTI油价 | tdx-connector + WebSearch |

**输出规则**：每个因子标注：
- 🔴 **结构性变化**（影响>3个月）：政策转向、利率周期逆转、贸易格局改变
- 🟡 **中期扰动**（影响1-3个月）：大盘回调、板块轮动、大额IPO
- ⚪ **短期噪音**（影响≤4周）：单日暴跌、体育赛事分流、例行会议

**核心判断**：本周宏观环境对主题持仓是「顺风/中性/逆风」？

🛑 **CHECKPOINT — Step 1.5 出口**：每个维度至少1个活跃因子，四个维度全扫后汇总为「整体宏观评级」（顺风/中性/逆风）。短期噪音占比>50%→主题逻辑未受侵蚀，维持仓位方向。

### Step 2：交叉验证

电话会信号 vs 紧缺度日报 → 有无背离？
- 电话会多次提及但紧缺度日报未达标 → 「主观预期 vs 数据滞后」
- 紧缺度日报强信号但电话会未提及 → 「自动化发现，人工待确认」

**信源**：聚宝盆（P0.5缓存）> 浑水调研（P0唯一IMA信源）。

### Step 2.5：主题SWOT + 风险收益比（v3.0新增）

基于本周拐点信号 + 宏观噪音判断，对核心主题做SWOT：

```markdown
## 主题SWOT

| 维度 | 内容 | 本周变化 |
|:---|:---|:---|
| **优势(S)** | 需求刚性(AI服务器+国产扩产) + 政策强(国产化专项) + 供给壁垒(认证/高纯) | → 维持 / ↗ 强化 / ↘ 弱化 |
| **劣势(W)** | 估值敏感 + 小市值波动大 + 信源依赖浑水单一来源 | → / ↗ / ↘ |
| **机会(O)** | FOMC后回调窗口 + 下半年扩产季 + 新催化剂 | → / ↗ / ↘ |
| **威胁(T)** | 地缘通胀 + 美联储超预期鹰派 + IPO持续吸血 + 新识别风险 | → / ↗ / ↘ |
```

**SWOT数据来源**：
- S/W → 从本周窄门筛股B/G维度提取汇总
- O → 从催化剂日历提取未来4周事件
- T → 从Step 1.5宏观噪音过滤器提取结构性变化

**风险收益比量化**：

```
中长期赔率 = Σ(核心标的G溢价加权) / Σ(核心标的估值风险加权)
短期赔率 = (上行催化剂密度 / 未来4周) / (宏观逆风强度)

中长期 ≥ 2:1 → 值得布局，短期仓位可忍受波动
中长期 < 2:1 → 降低仓位等待更好的入场点
短期 ≤ 1:1 → 收紧止损，暂不加仓
```

🛑 **CHECKPOINT — Step 2.5 出口**：中长期赔率<2:1 → 强制输出「降低仓位」建议。短期赔率<1:1→强制输出「收紧止损」建议。

### Step 3：操作建议

基于五个问题的结论：
- **加仓方向**：恶化+确认+正预期差 + 宏观中性/顺风
- **观察/减仓方向**：改善中 / 过度交易的 / 宏观逆风
- **下周电话会优先提问** 3-5个方向

## 动态宏观噪音过滤器补充说明

### 活跃因子识别方法
- **WebSearch**：搜索「本周 影响市场 资金流向 宏观 事件」
- **tdx-connector**：获取VIX/SOX/美债/汇率/原油实时数据
- **雪球/东方财富**：热帖关键词提取情绪信号
- **FOMC官方日历**：未来4周政策事件时间表

### 宏观评级规则
- **顺风(🔴)**：短期噪音为主+结构性因子偏正面+资金面宽松
- **中性(🟡)**：方向不明+多空交织+无极端事件
- **逆风(⚪)**：结构性恶化+资金面收紧+政策不确定性极高

### Step 1.6：宏观权重分配器（v3.1新增 — 盲区修正）

**核心理念**：宏观因子不是过滤条件，而是**权重分配器**。不同的宏观环境→钱流向不同类型的材料→应该重配不同类型的标的。

从 Step 1.5 的活跃因子输出，映射为材料类别权重调整：

#### 1.6.1 供给刚性类型与宏观环境映射

| 供给刚性类型 | 定义 | 受益宏观环境 | 受损宏观环境 |
|:---|:---|:---|:---|
| **产能约束型** | 扩产周期>18月，物理产能限制（如电子布、硅片） | 通胀上升、产能紧缺 | 利率飙升（capex成本↑） |
| **认证锁定型** | 客户认证>2年，新进入者壁垒（如WF6、前驱体） | 政策扶持、国产替代加速 | 地缘缓和、进口放松 |
| **地缘切割型** | 贸易管制/出口限制导致供给分裂（如稀土、高纯石英） | 地缘升温、出口管制扩大 | 中美缓和、管制放松 |

**使用方法**：Step 1.5 判断宏观环境顺风/逆风后→映射到上述三种类型的权重升降。

#### 1.6.2 动态权重分配表

每周输出时，生成当期权重分配表：

```markdown
## 宏观权重分配（YYYY-MM-DD 当期）

| 材料类别 | 基础权重 | 宏观调整 | 当期权重 | 代表品种 | 调整原因 |
|:---|:---:|:---:|:---:|:---|:---|
| 产能约束型 | 35% | ↗+5% | **40%** | 电子布、硅片 | 通胀维持+产能释放低于预期 |
| 认证锁定型 | 40% | → 0 | **40%** | WF6、前驱体 | 政策方向无变化 |
| 地缘切割型 | 25% | ↘-5% | **20%** | 稀土、高纯石英 | 中美关税窗口期暂未升级 |
```

**权重决策框架**（四维度→类型映射）：

| 宏观维度 | 当前方向 | → 产能约束型 | → 认证锁定型 | → 地缘切割型 |
|:---|:---|:---:|:---:|:---:|
| **资金流** | 流入/流出 | 流出→降10% | 流出→降5%（防御性强） | 流出→降10% |
| **情绪极值** | 贪婪/恐慌 | 恐慌→降5% | 恐慌→降5% | 恐慌→升10%（避险配置） |
| **政策/事件** | 确定/不确定 | 不确定→降5% | 不确定→升5%（政策催化预期） | 不确定→升10%（地缘溢价） |
| **跨市场联动** | 传导/隔离 | 强传导→降5% | 强传导→降5% | 强传导→升10%（独立逻辑） |

**调整规则**：
- 单维度调整±5%，四维度叠加±20%上限
- 当期权重 = 基础权重 + Σ四维度调整
- 基础权重默认为 产能约束型:认证锁定型:地缘切割型 = 35:40:25（基于锅师追踪品种分布）
- 每周围绕「当前什么在影响钱往哪流」重新判定四维度方向，不沿用上周调整

🛑 **CHECKPOINT — Step 1.6 出口**：权重分配表+四维度调整方向+调整原因必须全部输出。权重总和必须=100%。

### ⚠️ 动态性原则
- 每周围绕「当前什么在影响钱往哪流」重新扫描
- 不沿用上周因子清单——市场在变，重要的东西也在变
- 上周重要的因子本周可能退化为噪音，反之亦然
- **不固定宏观指标清单**——固定清单(12项)是日度监控用，周度方向判断必须动态
- 星大方法论参考：见 `深度研究/0615-0621/2026-06-21_星大方法论拆解与体系融入.md`
