# 选金 — 信息分级筛选器

不做信息仓库，只做金矿筛选。对每条输入信息输出判定：**存** 还是 **不存**。

---

## 判断铁律（一条命中即为"不存"）

| 铁律 | 判定 |
|:---|:---:|
| 能用 WebSearch 5 秒搜到 | ❌ 不存 |
| 时效 ≤ 24 小时（行情/快讯/当天新闻） | ❌ 不存 |
| 纯数据更新（CPI/PMI/利率/汇率等月度指标） | ❌ 不存 |
| 重复已有知识库内容 | ❌ 不存 |
| 未经信源验证的传闻（D/E 级） | ❌ 不存 |

---

## 值得沉淀的四类"金子"

| 类别 | 典型内容 | 为什么存 |
|:---|:---|:---|
| **产业链深度分析** | 卡点定位/壁垒排序/竞争格局拆解 | 经验性，不可搜索复现 |
| **S级紧缺信号** | 供给断崖/定价权突变/需求跳升 | 产业级拐点，错过不可逆 |
| **方法论/框架** | 研究管线/评分体系/信源规则 | 个人知识体系，不可替代 |
| **关联方穿透** | 股权结构/亲属网络/隐性关联 | 需要多源交叉推理，搜索不出 |

---

## 分级流程

```
输入信息
  │
  ├─ ① 能用搜索秒查？（判定≤5秒） ──── 是 ──→ ❌ 不存
  │
  ├─ ② 时效<24h？（判定≤3秒） ─────── 是 ──→ ❌ 不存
  │
  ├─ ③ 纯数据更新？（判定≤5秒） ───── 是 ──→ ❌ 不存
  │
  ├─ ④ 重复已有知识库？（查重≤10秒） ── 是 ──→ ❌ 不存
  │
  ├─ ⑤ 信源等级？（判定≤5秒） ─── E级 ──→ 🛑 STOP拦截
  │
  ├─ ⑥ 能满足S级信号标准？（判定≤15秒）
  │     │
  │     是 ──→ ✅ 存（金子，上 IMA）
  │     否 ──→ 是否有方法论/经验价值？
  │              │
  │              是 ──→ ✅ 存（本地）
  │              否 ──→ ❌ 不存
```

**总判定时间预算**：单条信息 ≤ 45秒。超时则标注「判定超时，默认不存（可手动覆写）」。

🔴 **CHECKPOINT — 铁律逐条判断**：每条铁律必须按顺序判断，命中即停。不可跳过中间铁律直接判断后续。

🔴 **CHECKPOINT — S级信号判定**：进入S级信号判定前，必须确认已通过前5条铁律筛选（未命中"不存"或"STOP"）。

🛑 **STOP**：遇上E级信源（微信小作文/未证实传闻），无论内容多吸引人，一律拒存并标注"信源不可靠，已拦截"。

🔴 **CHECKPOINT — 输出前终审**：输出选金结果前，必须每条信息标注判定理由（命中哪条铁律/为什么是金子）。

## 实测示例

**示例1：行业快讯**
- 输入：「今天锂电板块涨停5家」
- 判定：①能WebSearch秒查 → ❌ 不存
- 理由：命中铁律①，时效性快讯

**示例2：S级信号**
- 输入：「英伟达亲自抢购HVLP铜箔，缺口42%，扩产周期18个月」
- 判定：①不能秒查 ②时效>24h ③非纯数据 ④知识库无重复 ⑤信源A/B级 ⑥满足S级（供给断崖）
- 结论：✅ 存（金子，上IMA）
- 理由：通过6条铁律筛选，命中S级信号标准

**示例3：方法论**
- 输入：「二维评分体系：景气度×供给刚性，双4★为极度紧缺」
- 判定：①不能秒查 ②非时效 ③非数据 ④无重复 ⑤信源B级 ⑥非S级但属方法论
- 结论：✅ 存（本地）
- 理由：通过铁律⑤，属方法论/经验价值

**示例4：E级传闻**
- 输入：「微信群传某公司即将被收购」
- 判定：⑤信源E级 → 🛑 STOP
- 结论：拒存，标注「信源不可靠，已拦截」

## S级信号标准（决定是否上IMA）

| 信号 | 标准 |
|:---|:---|
| 供给断崖 | 产能收缩30%+ 且扩产周期>18个月 |
| 定价权突变 | 成功提价20%+ 且下游接受 |
| 需求跳升 | 需求增速环比翻倍（多方确认） |
| 产业拐点 | 连续3次信号景气度方向一致反转 |

## 禁止行为（反例黑名单）

| # | 禁止项 | 原因 | 正确做法 |
|---|--------|------|----------|
| 1 | 数据类/快讯类信息沉淀到 IMA | 浪费配额 | WebSearch实时查 |
| 2 | 把管理工具（日报/周报/月报）当知识沉淀 | 管理工具≠知识 | 只存方法论/经验 |
| 3 | 未经验证的传闻进入本地知识库 | 信源不可靠 | E级信源一律拦截 |
| 4 | 重复内容多次存储 | 冗余 | 存储前查重 |
| 5 | 跳过铁律顺序直接判断S级 | 破坏筛选逻辑 | 必须逐条铁律判断 |
| 6 | 判定不给理由只给结论 | 无法追溯依据 | 每条必须标注命中铁律 |
