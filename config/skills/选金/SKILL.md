# 选金 — 信息分级筛选器

不做信息仓库，只做金矿筛选。对每条输入信息输出判定：**存** 还是 **不存**。

---

## 判断铁律（一条命中即为"不存"）

| 铁律 | 判定 |
|:---|:---:|
| 能用 WebSearch 5 秒搜到 | ❌ 不存 |
| 时效 ≤ 24 小时（行情/快讯/当天新闻） | ❌ 不存 |
| 纯数据更新（CPI/PMI/利率/汇率等月度指标） | ❌ 不存 |
| 重复已有知识库内容 | ❌ 不存 |
| 未经信源验证的传闻（D/E 级） | ❌ 不存 |

---

## 值得沉淀的四类"金子"

| 类别 | 典型内容 | 为什么存 |
|:---|:---|:---|
| **产业链深度分析** | 卡点定位/壁垒排序/竞争格局拆解 | 经验性，不可搜索复现 |
| **S级紧缺信号** | 供给断崖/定价权突变/需求跳升 | 产业级拐点，错过不可逆 |
| **方法论/框架** | 研究管线/评分体系/信源规则 | 个人知识体系，不可替代 |
| **关联方穿透** | 股权结构/亲属网络/隐性关联 | 需要多源交叉推理，搜索不出 |

---

## 分级流程

```
输入信息
  │
  ├─ 能用搜索秒查？ ──── 是 ──→ ❌ 不存
  │
  ├─ 时效<24h？ ─────── 是 ──→ ❌ 不存
  │
  ├─ 纯数据更新？ ───── 是 ──→ ❌ 不存
  │
  ├─ 能满足S级信号标准？
  │     │
  │     是 ──→ ✅ 存（金子，上 IMA）
  │     否 ──→ 是否有方法论/经验价值？
  │              │
  │              是 ──→ ✅ 存（本地）
  │              否 ──→ ❌ 不存
```

🔴 **CHECKPOINT**：输出选金结果前，必须每条信息标注判定理由（命中哪条铁律/为什么是金子）。

🛑 **STOP**：遇上E级信源（微信小作文/未证实传闻），无论内容多吸引人，一律拒存并标注"信源不可靠，已拦截"。

## 实战分类表

### ✅ 值得沉淀

| 信息类型 | 存储位置 | 示例 |
|:---|:---|:---|
| 产业链卡点+壁垒分析 | IMA聚宝盆 | HVLP铜箔表面处理良率92%的天花板 |
| S级紧缺信号 | IMA聚宝盆 | 英伟达亲自抢购HVLP铜箔，缺口42% |
| 产业链全环节对比 | IMA聚宝盆 | 锂电池9环节壁垒排序 |
| 紧缺度追踪总表 | 本地 | 12品种月度演进 |
| 方法论/框架 | 本地 Skill | 二维评分/信源六层体系 |
| 关联方穿透分析 | 本地 + IMA | 宁德时代氢能链隐山资本穿透 |

### ❌ 无需沉淀

| 信息类型 | 替代方式 | 示例 |
|:---|:---|:---|
| 每日行情/涨跌幅 | WebSearch 实时 | 今天锂电板块谁涨停 |
| 宏观月度数据 | 统计局/央行官网 | 5月CPI 4.2% |
| 新闻快讯 | 财联社/金十数据 | 霍尔木兹海峡关闭 |
| 单一卖方研报观点 | 东方财富研报中心 | 高盛上调铜冠铜箔目标价 |
| 随口讨论/碎片观点 | 无法验证 | 社群传闻/小作文 |
| 日/周/月报 | 本地工作记录 | 管理工具，不是知识 |

## S级信号标准（决定是否上IMA）

| 信号 | 标准 |
|:---|:---|
| 供给断崖 | 产能收缩30%+ 且扩产周期>18个月 |
| 定价权突变 | 成功提价20%+ 且下游接受 |
| 需求跳升 | 需求增速环比翻倍（多方确认） |
| 产业拐点 | 连续3次信号景气度方向一致反转 |

---

## 禁止行为

- 数据类/快讯类信息沉淀到 IMA（浪费配额）
- 把管理工具（日报/周报/月报）当知识沉淀
- 未经验证的传闻进入本地知识库
- 重复内容多次存储

---

## 全局检查点规则

分级流程必须按铁律顺序逐条判断，不可跳过。🛑 STOP 触发时必须输出拦截信息，不继续存储操作。所有判定必须有理由标注，不可只给结论不给依据。
