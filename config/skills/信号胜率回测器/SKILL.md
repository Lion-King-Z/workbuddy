# 信号胜率回测器 Skill v1.0

**定位**：投研信号的**事后裁判**。不预测——只回测过去，告诉你信号准不准。

**核心原则**：
1. 不回测=盲飞——不知道信号准不准，B/G/C权重就是拍脑袋
2. 回测周期：1周（短期）/ 2周（中期）/ 4周（中期）/ 12周（长期）
3. 胜率=收益>0的标的占比；超额收益=标的收益-同期沪深300收益
4. 数据积累3个月后，可以反过来优化B/G/C权重

---

## 一、触发条件

- 每周日21:30自动触发
- 用户问"信号准不准""胜率多少""回测一下"
- 积累足够数据后（≥4周），可做因子有效性分析

---

## 二、回测流程

### Step 0：读取历史窄门筛股报告

读取过去N周的每日窄门筛股报告（`深度研究/<周文件夹>/YYYY-MM-DD_每日窄门筛股.md`）：
- 提取每日Top5/Top10的：标的代码、综合分、B分、G分、C分、评级、买入建议
- 记录报告日期作为"信号产生日"

### Step 1：获取标的实际收益

对每个信号标的，通过tdx-connector获取：
- 信号产生日收盘价（基准价）
- 1周后收盘价 → 1周收益
- 2周后收盘价 → 2周收益
- 4周后收盘价 → 4周收益（如已过足够时间）

同时获取同期沪深300收益作为基准：
- 超额收益 = 标的收益 - 沪深300收益

### Step 2：胜率统计

| 统计维度 | 计算方式 |
|:---|:---|
| 整体胜率 | 收益>0的标的数 / 总标的数 |
| 超额胜率 | 超额收益>0的标的数 / 总标的数 |
| 平均收益 | 所有标的收益的算术平均 |
| 平均超额收益 | 所有标的超额收益的算术平均 |
| Top1胜率 | 排名第1的标的收益>0的比例 |
| Top5胜率 | 前5名标的收益>0的比例 |
| Top10胜率 | 前10名标的收益>0的比例 |
| 最优标的 | 区间收益最高的标的 |
| 最差标的 | 区间收益最低的标的 |

### Step 3：因子有效性分析（需≥4周数据）

| 因子 | 分析方法 | 判断 |
|:---|:---|:---|
| B（瓶颈强度） | 按B分高/中/低分组，比较各组平均收益 | B高组收益>B低组→B因子有效 |
| G（预期差） | 按G分高/中/低分组，比较各组平均收益 | G高组收益>G低组→G因子有效 |
| C（确定性） | 按C乘数高/低分组，比较各组平均收益 | C高组收益>C低组→C因子有效 |
| 评级 | 按强力买入/建议买入/可买入分组比较 | 强力买入组收益>可买入组→评级有效 |
| 黄金赔率 | 黄金赔率标的 vs 非黄金赔率标的 | 黄金赔率组收益更高→黄金赔率有效 |

### Step 4：权重优化建议（需≥12周数据）

当数据积累≥12周时，输出B/G/C权重优化建议：

```
当前权重：B×0.30 + G×0.40 + 催化剂×0.30

回测发现：
- B高组平均收益 X% vs B低组 Y% → B因子有效性：强/中/弱
- G高组平均收益 X% vs G低组 Y% → G因子有效性：强/中/弱
- 催化剂高组平均收益 X% vs 催化剂低组 Y% → 催化剂因子有效性：强/中/弱

建议权重调整：
- 若B因子有效性显著强于G → B权重+5%，G权重-5%
- 若催化剂因子无效 → 催化剂权重-5%，B或G+5%
- 若三因子有效性接近 → 维持当前权重
```

> ⚠️ 权重优化建议只做数据呈现，不自动修改Skill。由老大决定是否调整。

## 五、注意事项

- 回测需要标的在信号日后仍在交易（停牌/退市标的排除）
- 收益计算用收盘价，不考虑交易成本和滑点
- 回测结果受市场环境影响——牛市胜率高、熊市胜率低，需结合市场背景解读
- 12周以下数据不输出权重优化建议，避免样本不足
- 严格遵循决策边界：只做数据回测和建议，不自动修改Skill权重
