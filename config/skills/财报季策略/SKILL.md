# 财报季策略 Skill v1.0

**定位**：财报季前后的交易策略框架。专业团队的财报季交易手册。

**核心原则**：
1. 财报前必须做预期管理——市场预期多少？实际可能多少？
2. 财报后必须有三套预案——超预期/符合/不及预期各自怎么操作
3. 财报季是最大的催化剂，也是最大的风险点
4. 只提策略预案，不替老大做决定

---

## 一、触发条件

- 用户问"财报季""业绩预告""财报策略""该不该赌财报"
- 催化剂日历中有财报日期的标的
- 业绩预告/快报发布时

| 数据项 | 来源 | 用途 |
|:---|:---|:---|

| 历史业绩兑现率 | 年报/季报历史 | 公司业绩可信度 |

## 三、财报前预期管理

### 3.1 预期定位

| 市场状态 | 判断依据 | 含义 |
|:---|:---|:---|
| 预期偏高 | 一致预期近4周连续上修 + 股价已超T1 | 财报需大幅超预期才能维持 |
| 预期合理 | 一致预期稳定 + 股价在T1附近 | 财报符合预期即可 |
| 预期偏低 | 一致预期近4周连续下修 + 股价低于T1 | 财报超预期概率高 |
| 预期混乱 | 一致预期分歧大（上修下修并存） | 高不确定性 |

### 3.2 市场已定价评估

| 评估项 | 计算 | 判断 |
|:---|:---|:---|
| 股价 vs T1 | 当前价 / T1 | >100%→已定价乐观 / 80-100%→合理 / <80%→悲观 |
| 近20日涨幅 | tdx-connector | >20%→预期升温 / <-10%→预期降温 |
| PE(TTM) vs 行业 | pe-valuation-analyst | >行业均值20%→偏高 |
| 券商覆盖数变化 | WebSearch | 近期新增覆盖→关注度上升 |

### 3.3 财报前仓位建议

| 预期定位 | 股价位置 | 仓位建议 | 理由 |
|:---|:---|:---|:---|
| 预期偏高 + 股价超T1 | 高位 | 减仓20-30% | 财报不及预期风险大 |
| 预期偏高 + 股价在T1 | 中位 | 持有，不加仓 | 赌财报风险收益不对等 |
| 预期合理 + 股价在T1 | 中位 | 持有 | 符合预期即可维持 |
| 预期偏低 + 股价低于T1 | 低位 | 可加仓20-30% | 超预期概率高，下行有限 |
| 预期混乱 | 任意 | 减仓至安全水平 | 高不确定性，等财报后再定 |

## 四、财报后反应预案

### 4.1 三种情景定义

| 情景 | 定义 | 判断标准 |
|:---|:---|:---|
| 🟢 超预期 | 实际EPS > 一致预期 × 1.10 | 超预期10%以上 + 指引上调 |
| 🟡 符合预期 | 一致预期 × 0.90 ≤ 实际EPS ≤ 一致预期 × 1.10 | 在预期范围内 |
| 🔴 不及预期 | 实际EPS < 一致预期 × 0.90 | 低于预期10%以上 或 指引下调 |

### 4.2 超预期情景操作预案

| 条件 | 操作 | 仓位调整 |
|:---|:---|:---|
| 超预期 + 指引上调 + 股价未大涨 | 🟢 加仓 | +20-30% |
| 超预期 + 指引上调 + 股价已大涨 | 🟡 持有 | 不加仓，等回调 |
| 超预期 + 指引持平 | 🟡 持有 | 观察后续指引 |
| 超预期 + 风险评级🟢 | 🟢 加仓 | +20% |

### 4.3 符合预期情景操作预案

| 条件 | 操作 | 仓位调整 |
|:---|:---|:---|
| 符合 + 指引上调 | 🟢 加仓 | +10-20% |
| 符合 + 指引持平 + 股价在T1 | 🟡 持有 | 不变 |
| 符合 + 指引持平 + 股价超T2 | 🟡 减仓 | -20% |
| 符合 + 指引下调 | 🔴 减仓 | -30-50% |

### 4.4 不及预期情景操作预案

| 条件 | 操作 | 仓位调整 |
|:---|:---|:---|
| 不及预期 + 指引下调 + 基本面未破 | 🔴 减仓 | -30-50% |
| 不及预期 + 指引下调 + 瓶颈证伪 | 🔴 止损 | 清仓 |
| 不及预期 + 风险评级升至🔴 | 🔴 止损 | 清仓 |
| 不及预期 + 财务健康恶化 | 🔴 止损 | 清仓 |
| 不及预期 + 但长期逻辑未变 + 股价大跌 | 🟡 观望 | 等企稳再定 |

## 五、业绩兑现率历史检查

对有历史业绩数据的标的，检查过去4个季度的业绩兑现率：

| 季度 | 一致预期EPS | 实际EPS | 兑现率 | 超预期/不及 |
|:---|:---:|:---:|:---:|:---|
| Q-4 | X.XX | X.XX | XX% | 超预期/符合/不及 |
| Q-3 | X.XX | X.XX | XX% | ... |
| Q-2 | X.XX | X.XX | XX% | ... |
| Q-1 | X.XX | X.XX | XX% | ... |

**兑现率趋势**：
- 连续4季度超预期 → 公司业绩可信度高，可赌财报
- 兑现率波动大 → 不可预测，建议减仓避险
- 连续2季度不及预期 → 业绩趋势恶化，建议财报前减仓

## 七、注意事项

- 财报季策略是预案，不是预测——不预测财报数据，只做"如果...则..."的情景规划
- 一致预期可能快速变化，财报前1周需重新确认
- 业绩指引比当季业绩更重要——指引上调=未来预期上修
- 非财报季时本Skill不主动触发
- 严格遵循决策边界：只提预案，不替老大执行
